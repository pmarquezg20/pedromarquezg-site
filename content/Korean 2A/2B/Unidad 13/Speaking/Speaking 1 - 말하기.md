## Macheteado para repetir
- 학교 근처로 이사하고 싶은데 괜찮은 집이 있을지 모르겠어요.
  - Quiero mudarme cerca de la escuela, pero no sé si habrá una casa buena.
  - Machete: 학교 / 근처로 / 이사하고 / 싶은데 / 괜찮은 / 집이 / 있을지 / 모르겠어요.
- 왜요? 지금 사는 집이 마음에 안 들어요?
  - ¿Por qué? ¿No le gusta la casa donde vive ahora?
  - Machete: 왜요? / 지금 / 사는 / 집이 / 마음에 / 안 / 들어요?
- 지금 사는 집은 방이 넓어서 좋기는 하지만 학교에서 너무 멀어요.
  - La casa donde vivo ahora tiene la habitación grande, lo cual está bien, pero está demasiado lejos de la escuela.
  - Machete: 지금 / 사는 / 집은 / 방이 / 넓어서 / 좋기는 / 하지만 / 학교에서 / 너무 / 멀어요.
- 그래요? 그럼 학교 근처에 있는 부동산에 한번 가보세요.
  - ¿Ah, sí? Entonces vaya a una agencia inmobiliaria cerca de la escuela.
  - Machete: 그래요? / 그럼 / 학교 / 근처에 / 있는 / 부동산에 / 한번 / 가보세요.
- 그렇지 않아도 수업 후에 가보려고요.
  - De hecho, pensaba ir después de clase.
  - Machete: 그렇지 / 않아도 / 수업 / 후에 / 가보려고요.
- 어떤 집을 구하는데요?
  - ¿Qué tipo de casa busca?
  - Machete: 어떤 / 집을 / 구하는데요?
- 시설이 잘 되어 있는 원룸을 구했으면 좋겠어요.
  - Me gustaría encontrar un estudio bien equipado.
  - Machete: 시설이 / 잘 / 되어 / 있는 / 원룸을 / 구했으면 / 좋겠어요.
- 요즘 새로 지은 원룸이 많으니까 쉽게 구할 수 있을 거예요.
  - Últimamente hay muchos estudios recién construidos, así que podrá encontrar uno fácilmente.
  - Machete: 요즘 / 새로 / 지은 / 원룸이 / 많으니까 / 쉽게 / 구할 / 수 / 있을 / 거예요.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

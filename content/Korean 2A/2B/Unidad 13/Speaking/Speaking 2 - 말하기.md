## Macheteado para repetir
- 어서오세요. 원룸을 하나 찾고 있는데요.
  - Bienvenido. Busco un estudio.
  - Machete: 어서오세요. / 원룸을 / 하나 / 찾고 / 있는데요.
- 언제쯤 이사하실 계획인가요?
  - ¿Cuándo tiene pensado mudarse?
  - Machete: 언제쯤 / 이사하실 / 계획인가요?
- 두 달 후에 집 계약이 끝나기 때문에 그 전에 이사했으면 좋겠어요.
  - El contrato de la casa termina dentro de dos meses, así que me gustaría mudarme antes de eso.
  - Machete: 두 / 달 / 후에 / 집 / 계약이 / 끝나기 / 때문에 / 그 / 전에 / 이사했으면 / 좋겠어요.
- 아, 그러세요?
  - ¿Ah, sí?
  - Machete: 아, / 그러세요?
- 마침 좋은 원룸이 하나 있는데, 근처에 공원도 있고 주변이 조용해서 살기 좋아요.
  - Justo hay un buen estudio; hay un parque cerca y los alrededores son tranquilos, así que es agradable para vivir.
  - Machete: 마침 / 좋은 / 원룸이 / 하나 / 있는데, / 근처에 / 공원도 / 있고 / 주변이 / 조용해서 / 살기 / 좋아요.
- 집세는 어떻게 돼요?
  - ¿Cuánto es el alquiler?
  - Machete: 집세는 / 어떻게 / 돼요?
- 전세는 7천만 원이고, 월세는 보증금 천만 원에 50만 원이에요.
  - En arrendamiento son 70 millones de wones, y en alquiler mensual son 10 millones de depósito más 500.000 wones al mes.
  - Machete: 전세는 / 7천만 / 원이고, / 월세는 / 보증금 / 천만 / 원에 / 50만 / 원이에요.
- 좀 비싸네요.
  - Es un poco caro.
  - Machete: 좀 / 비싸네요.
- 좀 비싸기는 하지만 깨끗하고 시설도 잘 되어 있기 때문에 마음에 드실 거예요.
  - Está un poco caro, pero está limpio y bien equipado, así que le gustará.
  - Machete: 좀 / 비싸기는 / 하지만 / 깨끗하고 / 시설도 / 잘 / 되어 / 있기 / 때문에 / 마음에 / 드실 / 거예요.
- 한번 구경해 보세요.
  - Échele un vistazo.
  - Machete: 한번 / 구경해 / 보세요.
- 네, 그럼 보고 결정할게요.
  - Sí, lo veré y decidiré.
  - Machete: 네, / 그럼 / 보고 / 결정할게요.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

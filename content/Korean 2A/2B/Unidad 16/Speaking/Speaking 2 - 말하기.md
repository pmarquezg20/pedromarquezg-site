## Macheteado para repetir
- 이번 추석 연휴에 뭐 할 거야?
  - ¿Qué vas a hacer en estas vacaciones de Chuseok?
  - Machete: 이번 / 추석 / 연휴에 / 뭐 / 할 / 거야?
- 글쎄, 아직 잘 모르겠는데.
  - Bueno, todavía no lo sé bien.
  - Machete: 글쎄, / 아직 / 잘 / 모르겠는데.
- 그럼, 토요일에 반 친구들을 초대할까 하는데 너도 올래?
  - Entonces, el sábado estoy pensando en invitar a los compañeros de clase, ¿vienes tú también?
  - Machete: 그럼, / 토요일에 / 반 / 친구들을 / 초대할까 / 하는데 / 너도 / 올래?
- 재미있겠다. 나도 갈게.
  - Suena divertido. Iré.
  - Machete: 재미있겠다. / 나도 / 갈게.
- 내가 뭐 도와줄 거 있어?
  - ¿Puedo ayudarte en algo?
  - Machete: 내가 / 뭐 / 도와줄 / 거 / 있어?
- 그럼, 음식 준비하는 것 좀 도와줄래?
  - Entonces, ¿me puedes ayudar a preparar la comida?
  - Machete: 그럼, / 음식 / 준비하는 / 것 / 좀 / 도와줄래?
- 그래. 뭘 만들 건데?
  - De acuerdo. ¿Qué vas a preparar?
  - Machete: 그래. / 뭘 / 만들 / 건데?
- 송편을 만들까 하는데 해본 적이 없어서 잘할 수 있을지 모르겠어.
  - Estoy pensando en hacer songpyeon, pero como no lo he hecho nunca, no sé si podré hacerlo bien.
  - Machete: 송편을 / 만들까 / 하는데 / 해본 / 적이 / 없어서 / 잘할 / 수 / 있을지 / 모르겠어.
- 나 송편 만들 줄 알아. 내가 도와줄 테니까 걱정하지 마.
  - Yo sé hacer songpyeon. Te ayudaré, así que no te preocupes.
  - Machete: 나 / 송편 / 만들 / 줄 / 알아. / 내가 / 도와줄 / 테니까 / 걱정하지 / 마.
- 잘됐다. 그럼 내가 장을 미리 봐 놓을게.
  - Qué bien. Entonces compraré los ingredientes con antelación.
  - Machete: 잘됐다. / 그럼 / 내가 / 장을 / 미리 / 봐 / 놓을게.
- 그래. 그럼 토요일에 보자.
  - De acuerdo. Entonces nos vemos el sábado.
  - Machete: 그래. / 그럼 / 토요일에 / 보자.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

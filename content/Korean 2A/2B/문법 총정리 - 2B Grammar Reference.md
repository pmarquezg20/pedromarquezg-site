## 2. Por función

**Cuando quiero pedir permiso o prohibir**
- [[Unidad 14/Grammar/V-아도 어도 되다|V-아/어도 되다]] — se puede
- [[Unidad 14/Grammar/V-(으)면 안 되다|V-(으)면 안 되다]] — no se puede

**Cuando quiero hablar de cambios**
- [[Unidad 15/Grammar/A-아지다 어지다|A-아/어지다]] — cambió un estado (adjetivo: 좋아졌어요)
- [[Unidad 15/Grammar/V-게 되다|V-게 되다]] — cambió una situación (verbo: 먹게 됐어요)

**Cuando quiero ordenar en el tiempo**
- [[Unidad 15/Grammar/V-기 전에|V-기 전에]] — antes de
- [[Unidad 15/Grammar/V-(으)ㄴ 후에|V-(으)ㄴ 후에]] — después de
- [[Unidad 14/Grammar/A V-았을 었을 때|A/V-았을/었을 때]] — cuando (pasado)
- [[Unidad 18/Grammar/V-(으)ㄴ 지|V-(으)ㄴ 지]] — hace cuánto que

**Cuando quiero dar causa/razón**
- [[Unidad 13/Grammar/A V-기 때문에 N(이)기 때문에|A/V-기 때문에]] — causa (escrito/formal)
- [[Unidad 16/Grammar/A V-(으)ㄹ 테니까|A/V-(으)ㄹ 테니까]] — razón + intención/suposición

**Cuando quiero contrastar o conceder**
- [[Unidad 13/Grammar/A V-기는 하지만|A/V-기는 하지만]] — sí, X, pero Y
- [[Unidad 10/Grammar/A-(으)ㄴ데 V-는데 N인데 2|A-(으)ㄴ데 / V-는데]] — contexto/contraste suave

**Cuando quiero sugerir o expresar intención**
- [[Unidad 11/Grammar/V-는 게 어때요?|V-는 게 어때요?]] — ¿qué tal si...?
- [[Unidad 10/Grammar/V-(으)ㄹ래요|V-(으)ㄹ래요]] — intención/propuesta informal
- [[Unidad 11/Grammar/V-기로 하다|V-기로 하다]] — lo decidí
- [[Unidad 16/Grammar/V-(으)ㄹ까 하다|V-(으)ㄹ까 하다]] — pienso / estoy por (tentativo)

**Cuando quiero hablar de experiencia**
- [[Unidad 14/Grammar/V-(으)ㄴ 적(이) 있다 없다|V-(으)ㄴ 적(이) 있다/없다]] — he/nunca he

**Cuando quiero describir estados resultantes**
- [[Unidad 16/Grammar/V-아 어 놓다|V-아/어 놓다]] — dejé algo hecho (yo lo hice)
- [[Unidad 17/Grammar/V-아 어 있다|V-아/어 있다]] — está en ese estado (sin agente)

**Cuando quiero comparar o describir cómo es alguien/algo**
- [[Unidad 12/Grammar/A-아 어 보이다|A-아/어 보이다]] — se ve / parece
- [[Unidad 12/Grammar/A-(으)ㄴ 편이다 V-는 편이다|A-(으)ㄴ 편이다]] — más bien / tiende a
- [[Unidad 12/Grammar/N처럼 같이|N처럼 / 같이]] — como N
- [[Unidad 12/Grammar/A-게|A-게]] — adverbializador

**Irregulares (불규칙) — repásalos juntos**
- [[Unidad 11/Grammar/'ㅅ' 불규칙|'ㅅ' 불규칙]]
- [[Unidad 17/Grammar/'ㅎ' 불규칙|'ㅎ' 불규칙]]

## 3. Pares que se confunden (⚠️ ojo aquí)

| Confusión | Diferencia clave | Notas |
|---|---|---|
| **-아/어지다 vs -게 되다** | 지다 = adjetivos (estado); 게 되다 = verbos (situación) | [[Unidad 15/Grammar/A-아지다 어지다\|A-아/어지다]] · [[Unidad 15/Grammar/V-게 되다\|V-게 되다]] |
| **-아/어 놓다 vs -아/어 있다** | 놓다 = yo lo dejé hecho (transitivo); 있다 = está así (intransitivo, sin agente) | [[Unidad 16/Grammar/V-아 어 놓다\|V-아/어 놓다]] · [[Unidad 17/Grammar/V-아 어 있다\|V-아/어 있다]] |
| **-기로 하다 vs -(으)ㄹ까 하다 vs -(으)ㄹ래요** | 기로 하다 = ya decidido; (으)ㄹ까 하다 = tentativo; (으)ㄹ래요 = intención informal | [[Unidad 11/Grammar/V-기로 하다\|V-기로 하다]] · [[Unidad 16/Grammar/V-(으)ㄹ까 하다\|V-(으)ㄹ까 하다]] · [[Unidad 10/Grammar/V-(으)ㄹ래요\|V-(으)ㄹ래요]] |
| **-기 때문에 vs -아/어서 vs -(으)니까** | 기 때문에 = causa escrita/enfática; 아/어서 = neutra; (으)니까 = razón + orden/sugerencia | [[Unidad 13/Grammar/A V-기 때문에 N(이)기 때문에\|A/V-기 때문에]] |
| **-기는 하지만 vs -지만 vs -는데** | 기는 하지만 = concesión enfática; 지만 = contraste plano; 는데 = contexto suave | [[Unidad 13/Grammar/A V-기는 하지만\|A/V-기는 하지만]] · [[Unidad 10/Grammar/A-(으)ㄴ데 V-는데 N인데 2\|A-(으)ㄴ데/V-는데]] |
| **반말 vs 한다체** | 반말 = informal HABLADO (가, 방학이야); 한다체 = escrito/narrativo (간다, 방학이다) | [[Unidad 10/Grammar/반말\|반말]] · [[Unidad 18/Grammar/A-다 V-ㄴ다 는다 N(이)다\|한다체]] |
| **-(으)ㄴ 적 있다 vs -(으)ㄴ 지** | 적 = experiencia ("he hecho"); 지 = tiempo transcurrido ("hace X que") | [[Unidad 14/Grammar/V-(으)ㄴ 적(이) 있다 없다\|V-(으)ㄴ 적]] · [[Unidad 18/Grammar/V-(으)ㄴ 지\|V-(으)ㄴ 지]] |

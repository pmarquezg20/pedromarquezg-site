# Grammar 4 - A-게

Unidad: [[../Index - Unidad 12|Unidad 12: 저는 좀 조용한 편이에요]]
Fuente: Student Book 2B + [[../Grammar Audio/Track 26 - 문법|Track 26 · 문법]].

## Idea central
`A-게` convierte un adjetivo en la manera en que haces una accion.

## La idea clave
La idea clave: `A-게` no describe como ES una cosa; describe COMO haces la accion.

- `머리가 짧아요` = el pelo es corto.
- `머리를 짧게 잘랐어요` = corte el pelo de manera corta / me lo corte corto.

## Track 26 - ejemplo principal

| Coreano         | Espanol                                   | Como pensarlo                               |
| --------------- | ----------------------------------------- | ------------------------------------------- |
| 머리를 잘랐네요?       | ¿Te cortaste el pelo?                     | `자르다 -> 잘랐네요` nota un cambio.               |
| 네, 더워서 짧게 잘랐어요. | Si, me lo corte corto porque hacia calor. | `짧게` explica la manera/resultado de cortar. |

Desglose:

`더워서` = porque hacia calor.

`짧게` = corto / de manera corta.

`잘랐어요` = lo corte.

`짧게 잘랐어요` = lo corte corto.

## Forma

`A raiz + 게`

| Adjetivo | A-게 | Ejemplo |
|---|---|---|
| 짧다 | 짧게 | 머리를 짧게 잘랐어요. |
| 길다 | 길게 | 머리를 길게 길렀어요. |
| 깨끗하다 | 깨끗하게 | 방을 깨끗하게 청소했어요. |
| 조용하다 | 조용하게 | 조용하게 말해 주세요. |
| 예쁘다 | 예쁘게 | 사진을 예쁘게 찍었어요. |
| 크게 | 크게 | 크게 말해 주세요. |
| 작다 | 작게 | 이름을 작게 썼어요. |

Ojo: con `길다`, la forma es `길게`, no `기게`.

## Como pensarlo
Pregunta: ¿la palabra describe una cosa o la manera de hacer una accion?

### Describe cosa/estado
- `머리가 짧아요` = el pelo es corto.
- `방이 깨끗해요` = el cuarto esta limpio.
- `목소리가 커요` = la voz es fuerte/grande.

### Describe manera/resultado de accion
- `머리를 짧게 잘랐어요` = corte el pelo corto.
- `방을 깨끗하게 청소했어요` = limpie el cuarto limpiamente / lo deje limpio.
- `크게 말해 주세요` = hable fuerte, por favor.

## No confundir con V-는 게

Son dos `게` diferentes.

| Forma | Funcion | Ejemplo |
|---|---|---|
| `A-게` | manera de hacer accion | `짧게 잘랐어요` = lo corte corto |
| `V-는 게` | `것이`, una accion como sustantivo | `쉬는 게 어때요?` = ¿que tal descansar? |

Machete: si antes hay adjetivo, probablemente es `A-게`. Si antes hay verbo con `-는`, probablemente es `V-는 게`.

## Frases tuyas para producir

- `한국어를 천천히 말하려고 노력해요.` = Intento hablar coreano despacio.
- `선생님, 글씨 좀 크게 써 주세요.` = Profesor/a, escriba la letra un poco grande, por favor.
- `글씨를 크게 써 주세요.` = Escriba la letra grande, por favor.
- `사진을 예쁘게 찍고 싶어요.` = Quiero tomar la foto bonita.
- `머리를 짧게 자르면 어려 보여요.` = Si te cortas el pelo corto, te ves joven.

## Errores faciles

### 1. No uses adjetivo normal cuando quieres modificar la accion
- Correcto: `머리를 짧게 잘랐어요`.
- Raro: `머리를 짧은 잘랐어요`.

### 2. No confundas `짧게` y `짧은`
- `짧은 머리` = pelo corto. Modifica un sustantivo.
- `짧게 자르다` = cortar corto. Modifica un verbo.

### 3. No todo se traduce con “-mente”
En espanol a veces sale natural con adverbio: `깨끗하게` = limpiamente.
Pero a veces sale mejor con resultado: `짧게 자르다` = cortar corto.

## Mini machete para clase

1. Busca el verbo principal: `자르다`, `말하다`, `청소하다`, `쓰다`.
2. Pregunta: ¿como se hace esa accion?
3. Pon el adjetivo antes en forma `A-게`.

`짧다 + 자르다 -> 짧게 자르다`

## Macheteado para repetir
- 잘 듣고 따라해 보세요.
  - Escuche bien y repita.
  - Machete: 잘 / 듣고 / 따라해 / 보세요.
- 1. 동생과 저는 닮지 않았어요.
  - 1. Mi hermano menor y yo no nos parecemos.
  - Machete: 1. / 동생과 / 저는 / 닮지 / 않았어요.
- 2. 우리 할아버지는 참 젊게 사세요.
  - 2. Mi abuelo vive de una manera muy joven.
  - Machete: 2. / 우리 / 할아버지는 / 참 / 젊게 / 사세요.
- 3. 아이는 부모를 닮는 것 같아요.
  - 3. El niño parece parecerse a sus padres.
  - Machete: 3. / 아이는 / 부모를 / 닮는 / 것 / 같아요.
- 4. 켈리 씨는 누구를 닮았어요?
  - 4. ¿A quién se parece Kelly?
  - Machete: 4. / 켈리 / 씨는 / 누구를 / 닮았어요?
- 눈은 아빠를 닮고 코는 엄마를 닮았어요.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

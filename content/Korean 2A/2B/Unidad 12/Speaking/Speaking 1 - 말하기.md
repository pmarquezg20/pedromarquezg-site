## Macheteado para repetir
- 여기가 내 방이야.
  - Este es mi cuarto.
  - Machete: 여기가 / 내 / 방이야.
- 책이 많네요.
  - Tienes muchos libros.
  - Machete: 책이 / 많네요.
- 이건 친구들 사진이에요?
  - ¿Esta es una foto con tus amigos?
  - Machete: 이건 / 친구들 / 사진이에요?
- 응. 작년 크리스마스 때 찍었어.
  - Sí. La sacamos en la Navidad del año pasado.
  - Machete: 응. / 작년 / 크리스마스 / 때 / 찍었어.
- 참 즐거워 보여요.
  - Parecen muy alegres.
  - Machete: 참 / 즐거워 / 보여요.
- 언니 옆에 있는 사람은 누구예요?
  - ¿Quién es la persona que está al lado de tu hermana?
  - Machete: 언니 / 옆에 / 있는 / 사람은 / 누구예요?
- 나하고 제일 친한 친구야.
  - Es mi mejor amiga.
  - Machete: 나하고 / 제일 / 친한 / 친구야.
- 어릴 때부터 자매처럼 지냈어.
  - Hemos sido como hermanas desde pequeñas.
  - Machete: 어릴 / 때부터 / 자매처럼 / 지냈어.
- 그래요?
  - ¿Ah, sí?
  - Machete: 그래요?
- 그래요? 언니하고 얼굴도 닮은 것 같아요.
  - ¿Ah, sí? Parece que se parecen.
  - Machete: 그래요? / 언니하고 / 얼굴도 / 닮은 / 것 / 같아요.
- 그런 말 많이 들어.
  - Me lo dicen mucho.
  - Machete: 그런 / 말 / 많이 / 들어.
- 우리 둘 다 이마가 넓고 눈이 커서 그런 것 같아.
  - Creo que es porque las dos tenemos la frente ancha y los ojos grandes.
  - Machete: 우리 / 둘 / 다 / 이마가 / 넓고 / 눈이 / 커서 / 그런 / 것 / 같아.
- 성격도 비슷해요?
  - ¿También tienen personalidades parecidas?
  - Machete: 성격도 / 비슷해요?
- 아니. 성격은 정말 달라.
  - No. La personalidad es muy diferente.
  - Machete: 아니. / 성격은 / 정말 / 달라.
- 나는 활발한 성격인데 내 친구는 내성적이야.
  - Yo soy extrovertida, pero mi amiga es introvertida.
  - Machete: 나는 / 활발한 / 성격인데 / 내 / 친구는 / 내성적이야.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

## Macheteado para repetir
- 이 근처에 있는 식당 중에서 어디가 제일 좋아?
  - De los restaurantes que hay por aquí, ¿cuál es el mejor?
  - Machete: 이 / 근처에 / 있는 / 식당 / 중에서 / 어디가 / 제일 / 좋아?
- 식당은 왜?
  - ¿Para qué necesitas un restaurante?
  - Machete: 식당은 / 왜?
- 다음 주 토요일이 우리 반 친구 생일이라서 친구들하고 같이 저녁 먹으려고.
  - El próximo sábado es el cumpleaños de un compañero de clase y quiero cenar con los amigos.
  - Machete: 다음 / 주 / 토요일이 / 우리 / 반 / 친구 / 생일이라서 / 친구들하고 / 같이 / 저녁 / 먹으려고.
- 친구들이 어떤 음식을 좋아하는데?
  - ¿Qué comida les gusta a los amigos?
  - Machete: 친구들이 / 어떤 / 음식을 / 좋아하는데?
- 다 잘 먹는데 친구들 중에 매운 음식을 잘 못 먹는 사람이 있어.
  - Comen de todo, pero hay alguien entre los amigos que no puede comer comida picante.
  - Machete: 다 / 잘 / 먹는데 / 친구들 / 중에 / 매운 / 음식을 / 잘 / 못 / 먹는 / 사람이 / 있어.
- 그럼 사거리에 있는 서울 식당이 어때?
  - Entonces, ¿qué tal el restaurante Seúl en el cruce?
  - Machete: 그럼 / 사거리에 / 있는 / 서울 / 식당이 / 어때?
- 서울 식당은 갈비가 유명한데 다른 음식도 다 맛있고 값도 비싸지 않아.
  - El restaurante Seúl es famoso por las costillas, pero toda la comida está buena y no es cara.
  - Machete: 서울 / 식당은 / 갈비가 / 유명한데 / 다른 / 음식도 / 다 / 맛있고 / 값도 / 비싸지 / 않아.
- 미리 예약을 해야 할까?
  - ¿Hay que reservar con antelación?
  - Machete: 미리 / 예약을 / 해야 / 할까?
- 주말에는 항상 사람이 많으니까 예약해야 될 거야.
  - Los fines de semana siempre hay mucha gente, así que tendrás que reservar.
  - Machete: 주말에는 / 항상 / 사람이 / 많으니까 / 예약해야 / 될 / 거야.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

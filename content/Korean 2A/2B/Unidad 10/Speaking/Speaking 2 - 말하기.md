## Macheteado para repetir
- 스티븐, 뭐 먹을래?
  - Steven, ¿qué vas a comer?
  - Machete: 스티븐, / 뭐 / 먹을래?
- 종류가 많네. 여긴 뭐가 맛있어?
  - Hay mucha variedad. ¿Qué está bueno aquí?
  - Machete: 종류가 / 많네. / 여긴 / 뭐가 / 맛있어?
- 이 식당은 삼계탕이 맛있는데 한번 먹어볼래?
  - Este restaurante es bueno para el samgyetang, ¿quieres probarlo?
  - Machete: 이 / 식당은 / 삼계탕이 / 맛있는데 / 한번 / 먹어볼래?
- 음, 난 닭고기는 좋아하는데 삼계탕은 별로 안 좋아해.
  - Bueno, me gusta el pollo, pero el samgyetang no me gusta mucho.
  - Machete: 음, / 난 / 닭고기는 / 좋아하는데 / 삼계탕은 / 별로 / 안 / 좋아해.
- 그래? 그럼 감자탕 어때? 안 먹어봤으면 한번 먹어봐.
  - ¿Ah, sí? Entonces, ¿qué tal el gamjatang? Si no lo has comido, pruébalo.
  - Machete: 그래? / 그럼 / 감자탕 / 어때? / 안 / 먹어봤으면 / 한번 / 먹어봐.
- 좋아. 그럼 감자탕 시키자.
  - De acuerdo. Entonces pidamos gamjatang.
  - Machete: 좋아. / 그럼 / 감자탕 / 시키자.
- 아주머니, 여기 감자탕 2인분 주세요.
  - Señora, dos raciones de gamjatang, por favor.
  - Machete: 아주머니, / 여기 / 감자탕 / 2인분 / 주세요.
- 맛이 어때? 입에 맞아?
  - ¿Qué tal está? ¿Está bueno?
  - Machete: 맛이 / 어때? / 입에 / 맞아?
- 응, 조금 매운데 맛있어.
  - Sí, está un poco picante pero está bueno.
  - Machete: 응, / 조금 / 매운데 / 맛있어.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

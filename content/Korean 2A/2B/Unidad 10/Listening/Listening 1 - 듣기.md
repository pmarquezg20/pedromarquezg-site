## Macheteado para repetir
- 잘 듣고 맞는 것을 고르세요.
  - Escuche bien y elija la respuesta correcta.
  - Machete: 잘 / 듣고 / 맞는 / 것을 / 고르세요.
- 벌써 2시야. 우리 점심 먹자.
  - Ya son las 2. Vamos a comer.
  - Machete: 벌써 / 2시야. / 우리 / 점심 / 먹자.
- 아, 벌써 그렇게 됐어?
  - ¿Ah, ya?
  - Machete: 아, / 벌써 / 그렇게 / 됐어?
- 그럼 우리 지금 바쁘니까 피자 시켜 먹을까?
  - Estamos ocupados ahora, ¿pedimos pizza?
  - Machete: 그럼 / 우리 / 지금 / 바쁘니까 / 피자 / 시켜 / 먹을까?
- 그래, 좋아.
  - Bien.
  - Machete: 그래, / 좋아.
- 무슨 피자 먹을래?
  - ¿Qué pizza quieres?
  - Machete: 무슨 / 피자 / 먹을래?
- 글쎄, 불고기 피자 어때?
  - Hmm, ¿qué tal la pizza bulgogi?
  - Machete: 글쎄, / 불고기 / 피자 / 어때?
- 난 불고기는 좋아하는데 불고기 피자는 별로 안 좋아해
  - Me gusta el bulgogi, pero la pizza bulgogi no me gusta mucho.
  - Machete: 난 / 불고기는 / 좋아하는데 / 불고기 / 피자는 / 별로 / 안 / 좋아해
- 그럼 치즈 피자 시키자 내가 주문할게
  - Entonces pidamos pizza de queso. Yo hago el pedido.
  - Machete: 그럼 / 치즈 / 피자 / 시키자 / 내가 / 주문할게

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

# Repaso 2026-07-08 - 중에, 중에서, 에서 + -는데 + vocab salud

Objetivo: estudiar rapido lo que causa confusion. Los 빽빽이 quedaron centralizados en Unidad 11.

## 1. 중에 / 중에서 / 에서

### Idea central
`중` = dentro de un grupo / entre varias opciones.

`N 중에` y `N 중에서` se usan cuando escoges, comparas o preguntas **entre un grupo**.

- `한국 음식 중에서 뭘 제일 좋아해요?` = Entre las comidas coreanas, ¿cuál te gusta más?
- `운동 중에서 야구를 가장 좋아해요.` = Entre los deportes, me gusta más el béisbol.
- `우리 반 친구들 중에 누가 제일 키가 커요?` = Entre los compañeros de clase, ¿quién es el más alto?

### 중에 vs 중에서
En muchos casos son intercambiables, pero el matiz cambia.

`중에서` = más claro cuando eliges/comparas dentro de una lista o grupo.

`중에` = más corto y natural en conversación, pero puede significar también “durante”.

Ejemplo importante:

- `운동 중에 전화가 왔어요.` = Me llamaron durante el ejercicio.
- `운동 중에서 축구가 제일 좋아요.` = Entre los deportes, el fútbol es el que más me gusta.

Por eso, si hay riesgo de confusión, usa `중에서`.

### 에서
`에서` normalmente es lugar de acción o punto de origen.

- `학교에서 공부해요.` = estudio en la escuela.
- `콜롬비아에서 왔어요.` = vine de Colombia.
- `우리 반에서 스티븐 씨가 키가 제일 커요.` = En nuestra clase, Steven es el más alto.

Pero con categorías como comida, películas, deportes, suena mejor `중에서`:

- `한국 음식에서 뭘 좋아해요?` suena raro.
- `한국 음식 중에서 뭘 좋아해요?` natural.

### Machete
Si puedes traducir “entre/de entre”, usa `중에서`.

Si puedes traducir “durante”, usa `중에`.

Si puedes traducir “en / desde” como lugar real, usa `에서`.

## 3. -는데 repaso de la foto

`-는데` conecta contexto + resultado/comentario/contraste.

Ejemplo:

`겨울인데 별로 춥지 않아요.`  
Es invierno, pero no hace mucho frío.

Respuestas recomendadas:

1. `시험 기간인데 공부를 많이 못 했어요.`  
2. `약을 먹었는데 아직도 아파요.`  
3. `점심을 많이 먹었는데 배고파요.`  
4. `친구한테 전화를 했는데 안 받았어요.`  
5. `한국어를 많이 공부했는데 아직 말하기가 어려워요.`

### Punto clave
`별로` casi siempre pide negativo:

- `별로 춥지 않아요.` = no hace mucho frío.
- `별로 안 좋아해요.` = no me gusta mucho.
- `별로 비싸지 않아요.` = no es muy caro.

## 4. Vocab salud p.48-52

### Síntomas
- `속이 안 좋다` = sentirse mal del estómago / tener náuseas.
- `몸살이 나다` = tener dolor corporal / malestar de gripa.
- `다리를 다치다` = lastimarse la pierna.
- `소화가 안되다` = tener mala digestión.
- `배탈이 나다` = tener daño de estómago.
- `토하다` = vomitar.
- `어지럽다` = estar mareado.
- `멀미를 하다` = marearse en transporte.
- `얼굴에 뭐가 나다` = salir algo en la cara.
- `입맛이 없다` = no tener apetito.
- `기운이 없다` = no tener energía.
- `잠을 잘 못 자다` = no dormir bien.

### Hospital / lugares
- `내과` = medicina interna.
- `안과` = oftalmología.
- `치과` = odontología.
- `피부과` = dermatología.
- `정형외과` = ortopedia.
- `이비인후과` = otorrino.

### Medicinas
- `감기약` = medicina para gripa.
- `두통약` = medicina para dolor de cabeza.
- `멀미약` = medicina para mareo.
- `소화제` = digestivo.
- `해열제` = antipirético / para bajar fiebre.
- `안약` = gotas para ojos.
- `파스` = parche medicinal.
- `연고` = ungüento.

### Verbos con medicinas
- `약을 먹다` = tomar medicina.
- `약을 넣다` = ponerse gotas.
- `파스를 붙이다` = ponerse un parche.
- `연고를 바르다` = aplicar ungüento.

## 5. ㅅ 불규칙 - lo mínimo

Algunos verbos con 받침 `ㅅ` pierden `ㅅ` cuando sigue vocal.

- `낫다` -> `나아요` / `나았어요`
- `붓다` -> `부어요` / `부었어요`
- `짓다` -> `지어요` / `지었어요`
- `젓다` -> `저어요` / `저었어요`

Ejemplos:

- `감기가 빨리 나았으면 좋겠어요.`
- `눈이 좀 부은 것 같아요.`
- `집은 30년 전에 지었어요.`
- `커피에 설탕을 넣었으니까 잘 저어서 드세요.`

## 6. N마다

`N마다` = cada N / todos los N.

- `날마다 한 시간씩 운동해요.` = Hago ejercicio una hora cada día.
- `아침마다 뉴스를 봐요.` = Veo noticias cada mañana.
- `나라마다 문화가 달라요.` = Cada país tiene una cultura diferente.
- `여덟 시간마다 드세요.` = Tómelo cada ocho horas.

## Extra urgente - 소개하겠습니다 / 평화라는 뜻입니다

### 소개하겠습니다

`소개하다` = presentar / introducir / recomendar un lugar hablando de él.

`소개하겠습니다` = `소개하 + 겠 + 습니다`

Aquí `-겠습니다` no es futuro normal tipo “mañana haré...”. En presentaciones se usa como **voy a... / procedo a...** de forma formal.

- `오늘은 파스를 소개하겠습니다.` = Hoy voy a presentar Paz.
- `설명하겠습니다.` = Voy a explicar.
- `시작하겠습니다.` = Voy a empezar.
- `말씀드리겠습니다.` = Se lo voy a decir / le hablaré de...

Machete: `-겠습니다` = “lo haré / voy a hacerlo” formal, con intención clara.

### 평화라는

Frase completa:

`파스는 스페인어로 평화라는 뜻입니다.`  
Paz significa “paz” en español.

Desglose:

- `평화` = paz
- `-라고 하다` = llamarse / decirse “...”.
- `평화라고 하는` = que se llama / que se dice “paz”.
- `평화라는` = versión contraída de `평화라고 하는`.

Entonces:

`평화라는 뜻` = el significado llamado “paz” / el significado de “paz”.

Más natural en español: “significa paz”.

Otros ejemplos:

- `사랑이라는 뜻입니다.` = Significa “amor”.
- `친구라는 말` = la palabra “amigo”.
- `김치라는 음식` = una comida llamada kimchi.

¿Por qué `라는` y no solo `평화 뜻`?

Porque estás diciendo que la palabra “Paz” tiene el significado llamado “평화”. `라는` funciona como “llamado / que significa / la palabra o concepto de”.

### 뜻입니다

`뜻` = significado / sentido.  
`입니다` = es.

`뜻입니다` literalmente es “es el significado”, pero en español natural decimos “significa”.

Desglose completo:

- `파스는` = Paz
- `스페인어로` = en español
- `평화라는` = llamado/significando “paz”
- `뜻입니다` = es el significado

Natural: **Paz significa “paz” en español.**

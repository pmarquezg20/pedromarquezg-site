## Macheteado para repetir
- 아키라 씨, 왜 이렇게 기운이 없어요?
  - Akira, ¿por qué está tan decaído?
  - Machete: 아키라 / 씨, / 왜 / 이렇게 / 기운이 / 없어요?
- 모르겠어요. 요즘 계속 피곤하고 입맛도 없어요.
  - No sé. Últimamente me siento cansado y sin apetito.
  - Machete: 모르겠어요. / 요즘 / 계속 / 피곤하고 / 입맛도 / 없어요.
- 그럼 운동을 좀 해보는 게 어때요?
  - Entonces, ¿qué tal si haces un poco de ejercicio?
  - Machete: 그럼 / 운동을 / 좀 / 해보는 / 게 / 어때요?
- 민수 씨는 건강을 위해서 특별히 하는 운동이 있어요?
  - Minsu, ¿tienes algún ejercicio especial que hagas para la salud?
  - Machete: 민수 / 씨는 / 건강을 / 위해서 / 특별히 / 하는 / 운동이 / 있어요?
- 네, 저는 아침마다 수영을 하는데 기분도 좋고 스트레스도 풀려요.
  - Sí, nado todas las mañanas y me siento bien y alivio el estrés.
  - Machete: 네, / 저는 / 아침마다 / 수영을 / 하는데 / 기분도 / 좋고 / 스트레스도 / 풀려요.
- 그래요? 어디에서 하는데요?
  - ¿En serio? ¿Dónde lo haces?
  - Machete: 그래요? / 어디에서 / 하는데요?
- 회사 근처에 있는 수영장에서 해요.
  - Lo hago en la piscina que hay cerca de la empresa.
  - Machete: 회사 / 근처에 / 있는 / 수영장에서 / 해요.
- 다음 주부터 지오 씨도 같이 하기로 했는데 아키라 씨도 같이 할래요?
  - La semana que viene Jio también va a empezar, ¿Akira también quiere venir?
  - Machete: 다음 / 주부터 / 지오 / 씨도 / 같이 / 하기로 / 했는데 / 아키라 / 씨도 / 같이 / 할래요?
- 네, 저도 시간 내서 한번 가볼게요.
  - Sí, sacaré tiempo y probaré a ir.
  - Machete: 네, / 저도 / 시간 / 내서 / 한번 / 가볼게요.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

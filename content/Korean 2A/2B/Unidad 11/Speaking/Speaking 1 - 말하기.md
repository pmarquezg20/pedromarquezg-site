## Macheteado util para repetir

### 1. 어떻게 오셨어요?
- Traduccion: ¿En qué puedo ayudarle?
- Forma base: `오다` = venir.
- `오셨어요` = vino, honorífico/polite.
- En farmacia/hospital no significa literalmente “¿cómo vino?”, sino “¿qué le pasa?” / “¿en qué puedo ayudarle?”.
- Machete: `어떻게 / 오셨어요?`

### 2. 배가 아파서 왔는데요.
- Traduccion: Vine porque me duele el estómago.
- `배` = abdomen/estómago.
- `아파서` = porque duele.
- `왔는데요` = vine, y... / vine porque..., dicho suave.
- Machete: `배가 / 아파서 / 왔는데요.`

### 3. 속도 안 좋으세요?
- Traduccion: ¿También tiene malestar estomacal?
- `속` = estómago interior / sensación digestiva.
- `도` = también.
- `안 좋다` = no estar bien.
- Machete: `속도 / 안 / 좋으세요?`

### 4. 네, 속도 안 좋고 토할 것 같아요.
- Traduccion: Sí, tengo malestar y siento que voy a vomitar.
- `-고` conecta: y.
- `토하다` = vomitar.
- `V-(으)ㄹ 것 같아요` = parece que / siento que voy a.
- Machete: `속도 안 좋고 / 토할 것 같아요.`

### 5. 언제부터 아프셨어요?
- Traduccion: ¿Desde cuándo le duele?
- `언제부터` = desde cuándo.
- `아프셨어요` = le dolió / estuvo enfermo, honorífico.
- Machete: `언제부터 / 아프셨어요?`

### 6. 어제 저녁부터 계속 아파요.
- Traduccion: Me duele continuamente desde ayer por la noche.
- `어제 저녁부터` = desde ayer por la noche.
- `계속` = continuamente / sigue.
- Machete: `어제 저녁부터 / 계속 / 아파요.`

### 7. 그럼 이 약을 한번 드셔 보세요.
- Traduccion: Entonces pruebe tomar este medicamento.
- `약` = medicina.
- `드시다` = comer/tomar honorífico de `먹다`.
- `V-아/어 보다` = probar hacer algo.
- `드셔 보세요` = pruebe tomarlo.
- Machete: `그럼 / 이 약을 / 한번 / 드셔 보세요.`

### 8. 식후 3번, 8시간마다 드시면 돼요.
- Traduccion: Tómelo 3 veces después de las comidas, cada 8 horas.
- `식후` = después de comer.
- `3번` = tres veces.
- `8시간마다` = cada ocho horas.
- `V-(으)시면 돼요` = basta con que / puede hacerlo así.
- Machete: `식후 3번 / 8시간마다 / 드시면 돼요.`

### 9. 네, 얼마지요?
- Traduccion: Sí, ¿cuánto es?
- `얼마` = cuánto.
- `-지요?` = forma suave de confirmar/preguntar.
- Machete: `네 / 얼마지요?`

### 10. 5000원이에요. 이 약을 이틀 정도 드셔 보시고 그래도 안 나으면 병원에 가세요.
- Traduccion: Son 5.000 wones. Tome este medicamento durante unos dos días y, si aun así no mejora, vaya al hospital.
- `5000원이에요` = son 5.000 wones.
- `이 약을` = esta medicina.
- `이틀 정도` = unos dos días / aproximadamente dos días.
- `드셔 보시고` = tómelo/probéelo y luego... Honorífico + `보다` + `-고`.
- `그래도` = aun así / incluso entonces.
- `안 나으면` = si no mejora. De `낫다` = recuperarse/mejorar. Ojo: `ㅅ 불규칙`: `낫다 -> 나아요 -> 나으면`.
- `병원에 가세요` = vaya al hospital.
- Machete: `5000원이에요 / 이 약을 / 이틀 정도 / 드셔 보시고 / 그래도 / 안 나으면 / 병원에 가세요.`

### 11. 네, 감사합니다. 안녕히 계세요.
- Traduccion: Sí, gracias. Que esté bien / adiós.
- `안녕히 계세요` se dice cuando tú te vas y la otra persona se queda.
- Machete: `네 / 감사합니다 / 안녕히 계세요.`

## Nota conectada - V-기로 하다

Esta gramatica aparece fuerte en la unidad y en Track 17:

`다음 주부터 지오 씨도 같이 하기로 했는데`

Traduccion: Desde la próxima semana, Jio también quedó en hacerlo con nosotros / decidió hacerlo juntos, y...

Desglose:

- `다음 주부터` = desde la próxima semana.
- `지오 씨도` = Jio también.
- `같이` = juntos.
- `하기로 했는데` = decidió hacer / quedó en hacer, y... / pero...

`V-기로 하다` = decidir hacer / quedar en hacer.

Ejemplos:

- `한국어를 매일 공부하기로 했어요.` = Decidí estudiar coreano todos los días.
- `친구하고 운동하기로 했어요.` = Quedé en hacer ejercicio con un amigo.
- `내일부터 일찍 자기로 했어요.` = Decidí dormir temprano desde mañana.

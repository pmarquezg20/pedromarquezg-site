# Grammar 2 - N마다

Unidad: [[../Index - Unidad 11|Unidad 11: 운동을 좀 해 보는 게 어때요?]]
Fuente: Student Book 2B + audios de la app koreano.

## Idea central
`N마다` significa cada N / todos los N. Sirve para frecuencia, repeticion o para decir que algo aplica a cada miembro de un grupo.

## La idea clave
La idea clave: `마다` reparte la accion por cada unidad: cada dia, cada persona, cada ocho horas. No necesita otra particula en medio.

## Forma
`N + 마다`

Se pega directo: `시간마다`, `아침마다`, `사람마다`, `해마다`.

## Como pensarlo
No le metas particula en la mitad. Es `8시간마다`, no `8시간에마다`.

`마다` ya trae la idea de cada/todo, asi que evita duplicar con `매`:

- Natural: `해마다` o `매년`.
- Evita: `매년마다`.

## Ejemplos base y variantes
| Coreano | Espanol |
|---|---|
| 8시간마다 드세요. | Tomelo cada 8 horas. |
| 아침마다 수영해요. | Nado cada manana. |
| 해마다 건강 검사를 받아요. | Me hago chequeo medico cada ano. |
| 사람마다 생각이 달라요. | Cada persona piensa diferente. |

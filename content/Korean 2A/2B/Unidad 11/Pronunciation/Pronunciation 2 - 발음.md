## Macheteado para repetir
- 잘 듣고 따라해 보세요.
  - Escuche bien y repita.
  - Machete: 잘 / 듣고 / 따라해 / 보세요.
- 1. 이 약은 한 번에 몇 알씩 먹어요?
  - 1. ¿Cuántas pastillas de este medicamento se toman cada vez?
  - Machete: 1. / 이 / 약은 / 한 / 번에 / 몇 / 알씩 / 먹어요?
- 2. 입 위에 나비가 앉았어요.
  - 2. Una mariposa se posó sobre la boca.
  - Machete: 2. / 입 / 위에 / 나비가 / 앉았어요.
- 3. 부엌 안으로 들어가 보세요.
  - 3. Entre a la cocina.
  - Machete: 3. / 부엌 / 안으로 / 들어가 / 보세요.
- 4. 어제 왜 안 왔어요?
  - 4. ¿Por qué no viniste ayer?
  - Machete: 4. / 어제 / 왜 / 안 / 왔어요?
- 5. 바빠서 못 왔어요.
  - 5. No pude venir porque estaba ocupado.
  - Machete: 5. / 바빠서 / 못 / 왔어요.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

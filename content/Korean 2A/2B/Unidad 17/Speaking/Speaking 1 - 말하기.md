## Macheteado para repetir
- 오늘은 다른 때보다 늦게 왔네요.
  - Hoy llegaste más tarde que otras veces.
  - Machete: 오늘은 / 다른 / 때보다 / 늦게 / 왔네요.
- 오늘 부모님이 고향으로 돌아가셔서 공항에 모셔다 드리고 왔어요.
  - Hoy mis padres se fueron a su ciudad natal, así que les llevé al aeropuerto.
  - Machete: 오늘 / 부모님이 / 고향으로 / 돌아가셔서 / 공항에 / 모셔다 / 드리고 / 왔어요.
- 아, 그래요? 잘 모셔다 드렸어요?
  - ¿Ah, de verdad? ¿Les llevaste bien?
  - Machete: 아, / 그래요? / 잘 / 모셔다 / 드렸어요?
- 네. 그런데 비행기를 놓칠 뻔했어요.
  - Sí. Pero casi perdemos el avión.
  - Machete: 네. / 그런데 / 비행기를 / 놓칠 / 뻔했어요.
- 왜요? 무슨 일 있었어요?
  - ¿Por qué? ¿Qué pasó?
  - Machete: 왜요? / 무슨 / 일 / 있었어요?
- 길이 너무 막혀서 공항에 늦게 도착했어요.
  - Había demasiado tráfico y llegamos tarde al aeropuerto.
  - Machete: 길이 / 너무 / 막혀서 / 공항에 / 늦게 / 도착했어요.
- 아침부터 바빴겠네요.
  - Debes haber estado muy ocupado desde por la mañana.
  - Machete: 아침부터 / 바빴겠네요.
- 네. 정말 정신이 없었어요.
  - Sí. Fue un caos total.
  - Machete: 네. / 정말 / 정신이 / 없었어요.
- 그래도 비행기를 안 놓쳐서 다행이에요.
  - De todas formas, menos mal que no perdisteis el avión.
  - Machete: 그래도 / 비행기를 / 안 / 놓쳐서 / 다행이에요.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

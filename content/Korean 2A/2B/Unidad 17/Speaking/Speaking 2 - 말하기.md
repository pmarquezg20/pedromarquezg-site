## Macheteado para repetir
- 어떻게 오셨습니까?
  - ¿En qué puedo ayudarle?
  - Machete: 어떻게 / 오셨습니까?
- 가방을 잃어버려서 왔는데요.
  - He venido porque perdí un bolso.
  - Machete: 가방을 / 잃어버려서 / 왔는데요.
- 어디서 잃어버리셨습니까?
  - ¿Dónde lo perdió?
  - Machete: 어디서 / 잃어버리셨습니까?
- 오늘 아침에 지하철 2호선에 놓고 내렸어요.
  - Esta mañana lo dejé en el metro de la línea 2.
  - Machete: 오늘 / 아침에 / 지하철 / 2호선에 / 놓고 / 내렸어요.
- 가방이 어떻게 생겼습니까?
  - ¿Cómo es el bolso?
  - Machete: 가방이 / 어떻게 / 생겼습니까?
- 까만색 배낭인데 하얀색 인형이 달려있어요.
  - Es una mochila negra con un muñeco blanco colgado.
  - Machete: 까만색 / 배낭인데 / 하얀색 / 인형이 / 달려있어요.
- 가방 안에 뭐가 들어있습니까?
  - ¿Qué hay dentro del bolso?
  - Machete: 가방 / 안에 / 뭐가 / 들어있습니까?
- 책 두 권하고 지갑이 들어있어요.
  - Hay dos libros y una cartera.
  - Machete: 책 / 두 / 권하고 / 지갑이 / 들어있어요.
- 잠깐만요. 그런 가방은 없는데요.
  - Espere un momento. No tenemos ningún bolso así.
  - Machete: 잠깐만요. / 그런 / 가방은 / 없는데요.
- 그래요? 꼭 찾아야 하는데 큰일 났네요.
  - ¿De verdad? Necesito encontrarlo sí o sí. Es un problema.
  - Machete: 그래요? / 꼭 / 찾아야 / 하는데 / 큰일 / 났네요.
- 찾으면 연락드릴 테니까 여기에 전화번호를 써놓고 가세요.
  - Si lo encontramos, le avisaremos. Deje su número de teléfono aquí y váyase.
  - Machete: 찾으면 / 연락드릴 / 테니까 / 여기에 / 전화번호를 / 써놓고 / 가세요.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

## Macheteado para repetir
- 잘 듣고 따라해 보세요.
  - Escuche bien y repita.
  - Machete: 잘 / 듣고 / 따라해 / 보세요.
- 1. 한라산에 가봤어요?
  - 1. ¿Has ido al monte Halla?
  - Machete: 1. / 한라산에 / 가봤어요?
- 2. 지하철이 버스보다 편리해요.
  - 2. El metro es más conveniente que el bus.
  - Machete: 2. / 지하철이 / 버스보다 / 편리해요.
- 3. 여기서 신림동까지 얼마나 걸려요?
  - 3. ¿Cuánto se tarda de aquí a Sillim-dong?
  - Machete: 3. / 여기서 / 신림동까지 / 얼마나 / 걸려요?
- 4. 아키라 씨 전화번호를 알면 좀 연락해 주세요.
  - 4. Si sabes el número de Akira, por favor contáctame.
  - Machete: 4. / 아키라 / 씨 / 전화번호를 / 알면 / 좀 / 연락해 / 주세요.
- 네, 제가 연락할게요.
  - Sí, yo me pondré en contacto.
  - Machete: 네, / 제가 / 연락할게요.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

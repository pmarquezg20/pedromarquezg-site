# Grammar 4 - V-아/어 있다

Unidad: [[../Index - Unidad 17|Unidad 17: 비행기를 놓칠 뻔했어요]]
Fuente: Student Book 2B (p.189 + Appendix p.253) + audios de la app koreano.

## La idea clave
`V-아/어 있다` = **el estado que queda DESPUÉS de que una acción terminó, y que sigue así**.

- `앉아 있어요` = está sentado (ya se sentó, y sigue sentado).
- `문이 열려 있어요` = la puerta está abierta (alguien la abrió, sigue abierta).
- `걸려 있는 옷` = la ropa que está colgada.

El contraste clave es con `-고 있다` (que ya conoces del 1B/2A):

| Forma | Foco | Ejemplo |
|---|---|---|
| `-고 있다` | la acción EN CURSO | 앉고 있어요 = se está sentando (a medio camino) |
| `-아/어 있다` | el estado RESULTANTE | 앉아 있어요 = está sentado |

En español ambos se traducen con "estar + gerundio/participio": "está sentándose" vs "está sentado". Esa misma diferencia.

## Forma
Se une a **verbos** (sobre todo intransitivos y pasivos). Misma elección de vocal que `-아/어 보다`:

| | ㅏ, ㅗ | 하다 | ㅓ, ㅜ, ㅣ ... |
|---|---|---|---|
| 동사 | 앉다 → 앉**아 있다** | 입원하다 → 입원**해 있다** | 열리다 → 열**려 있다** |

Verbos estrella para describir dónde/cómo están las cosas (lista del 연습1 del libro, todos pasivos/intransitivos):

| Verbo | Con -아/어 있다 | Significado |
|---|---|---|
| 놓이다 (ser puesto) | 놓여 있다 | está puesto/colocado |
| 열리다 (ser abierto) | 열려 있다 | está abierto |
| 걸리다 (ser colgado) | 걸려 있다 | está colgado |
| 달리다 (ser colgado/adjunto) | 달려 있다 | está colgado/prendido (un adorno, llavero) |
| 붙다 (pegarse) | 붙어 있다 | está pegado |
| 서다 (pararse) | 서 있다 | está de pie |
| 앉다 (sentarse) | 앉아 있다 | está sentado |
| 눕다 (acostarse) | 누워 있다 | está acostado |
| 들다 (entrar/contener) | 들어 있다 | está adentro/contenido |

## Appendix del libro (문법 해설)
**Significado (p.253):** indica que, tras terminar una acción, **su estado continúa** (어떤 행위가 끝난 후에 그 상태가 계속 지속됨).

**Combinación:** con raíces de verbo.

| | ㅏ, ㅗ | 하다 | ㅓ, ㅜ, ㅣ ... |
|---|---|---|---|
| 동사 | 앉다 → 앉아 있다 | 입원하다 → 입원해 있다 | 열리다 → 열려 있다 |

Ejemplos del appendix:

| Coreano | Español |
|---|---|
| 학생들이 교실에 앉아 있어요. | Los estudiantes están sentados en el aula. |
| 친구가 교통사고가 나서 지금 병원에 입원해 있어요. | Mi amigo tuvo un accidente y ahora está hospitalizado. |
| 저기 걸려 있는 옷을 좀 보여 주세요. | Muéstreme la ropa que está colgada allá. |
| 까만색 가방에 파란색 인형이 달려 있어요. | El bolso negro tiene colgado un muñeco azul. |

**Nota (주의) del appendix:** los verbos de **ponerse ropa** (`입다, 쓰다, 신다, 벗다`) NO usan `-아/어 있다` para el estado: usan **`-고 있다`** (que aquí significa "llevar puesto", no "estar poniéndose"):

| Coreano | Español |
|---|---|
| 모자를 쓰고 있는 사람이 스티븐 씨예요. | El que lleva gorra es Steven. |
| 집에서는 편안한 옷을 입고 있어요. | En casa llevo ropa cómoda. |
| 발이 아파서 신발을 벗고 있었어요. | Me dolían los pies, así que estaba descalzo. |

O sea: `*모자를 써 있어요` es incorrecto. Con ropa, `-고 있다` cubre los dos sentidos (poniéndose Y puesto); el contexto decide.

## Transitivo vs intransitivo: el detalle que confunde
`-아/어 있다` va con verbos **que no llevan objeto** (o pasivos). Si el verbo lleva objeto, el estado resultante se dice con `-아/어 놓다/두다` (Unidad 16):

- `창문이 열려 있어요.` = la ventana está abierta (열리다, pasivo → 있다)
- `창문을 열어 놓았어요.` = dejé la ventana abierta (열다, con objeto → 놓다)

Mismo resultado en el mundo, distinto punto de vista: 있다 describe la escena; 놓다/두다 dice que alguien la dejó así a propósito.

## Conexión con la unidad
Es LA gramática para describir objetos perdidos en el 말하기 2: color + patrón + `-아/어 있다`:
`까만색 배낭인데 하얀색 인형이 달려 있어요. 안에 책 두 권하고 지갑이 들어 있어요.` = Es una mochila negra con un muñeco blanco colgado; adentro hay dos libros y una billetera.

Fíjate también en el uso como modificador con `-는` (Unidad 12, `-는 것`): `걸려 있는 옷`, `서 있는 사람` — el estado convertido en "que está + participio".

## Errores típicos de hispanohablantes
- Usar `-고 있다` para estados de posición: `*문이 열리고 있어요` = la puerta se está abriendo (¡en movimiento!). Para "está abierta": `열려 있어요`.
- Usar `-아/어 있다` con ropa: `*안경을 써 있어요` → `안경을 쓰고 있어요`.
- Olvidar el pasivo: `*문이 열어 있어요` → el verbo debe ser el pasivo `열리다`: `열려 있어요`. Pares útiles: 열다/열리다, 걸다/걸리다, 놓다/놓이다.
- `살다` y `입원하다` van con `-아/어 있다` para residencia/estado: `병원에 입원해 있어요`.
- 들다: `가방에 지갑이 들어 있어요` = la billetera está dentro del bolso. Muy usado, apréndelo como bloque: `N이/가 들어 있다`.

## Machete para Pedro
**"-고 있다 = video, -아/어 있다 = foto."**

El video muestra la acción ocurriendo; la foto muestra cómo quedó todo:

- 앉아 있다 = sentado, 서 있다 = de pie, 누워 있다 = acostado
- 열려 있다 = abierto, 걸려 있다 = colgado, 붙어 있다 = pegado, 놓여 있다 = puesto, 달려 있다 = prendido, 들어 있다 = adentro
- EXCEPCIÓN ropa: 입고/쓰고/신고 있다 = llevar puesto

## Mini-drill
Describe la escena (foto, no video):
1. La ventana está abierta y hay un gato sentado en la silla. → `창문이 열려 있고 의자에 고양이가 앉아 있어요.`
2. En la pared hay un cuadro colgado. → `벽에 그림이 걸려 있어요.`
3. Adentro de la mochila hay una billetera. → `가방 안에 지갑이 들어 있어요.`
4. El que lleva lentes está de pie junto a la puerta. → `안경을 쓰고 있는 사람이 문 옆에 서 있어요.` (ropa → 고 있다; posición → 아/어 있다)

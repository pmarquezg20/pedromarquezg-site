# Grammar 3 - V-아도/어도 되다

Unidad: [[../Index - Unidad 14|Unidad 14: 여기서 사진을 찍어도 돼요?]]
Fuente: Student Book 2B (p. 122) + Appendix 문법 해설 (p. 246) + audios de la app koreano.

## La idea clave
`V-아도/어도 되다` = **permiso**: "se puede...", "está bien si...", "no pasa nada si...".

- Pregunta: `찍어도 돼요?` = ¿puedo/se puede sacar fotos?
- Respuesta afirmativa: `네, 찍어도 돼요.` = sí, se puede.
- Es LA frase que da título a la unidad: `여기서 사진을 찍어도 돼요?`

Ojo con `되다`: aquí no significa "convertirse en", sino "estar bien / ser aceptable". `돼요` es la conjugación de 되다 (되 + 어요 → 돼요).

También sirve para decir que algo **no es obligatorio**: `내일 내도 돼요` = puedes entregarlo mañana (no hace falta hoy). Es la respuesta natural a `-아/어야 돼요?` (¿tengo que...?).

## Forma
Se combina con **verbos**. La elección 아/어 es la misma de siempre (igual que en `-아/어요`, `-아/어서`, `-아/어 보다`):

| Última vocal | Regla | Ejemplo |
|---|---|---|
| ㅏ, ㅗ | + 아도 되다 | 가다 → **가도 되다** |
| 하다 | → 해도 되다 | 전화하다 → **전화해도 되다** |
| ㅓ, ㅜ, ㅣ … | + 어도 되다 | 찍다 → **찍어도 되다** |
| Irregular 으 | cae la 으 + 어 | 쓰다 → **써도 되다** |
| Vocal ㅏ con 받침 | + 아도 되다 | 앉다 → **앉아도 되다** |

Formalidad: `-아도 됩니다` (formal), `-아도 돼요` (estándar), `-아도 돼` (반말).

## Appendix del libro (문법 해설)
**Significado:** expresa **permiso o autorización** para hacer una acción ("'-아도/어도 되다' is used to express permission to do an action").

**Combinación:** con raíces de verbo.

| | ㅏ, ㅗ | 하다 | ㅓ, ㅜ, ㅣ … |
|---|---|---|---|
| 동사 | 가다 → **가도 되다** | 전화하다 → **전화해도 되다** | 찍다 → **찍어도 되다** |

Ejemplos del appendix:

| Coreano | Español |
|---|---|
| 이 일이 끝나면 집에 가도 돼요. | Cuando termine este trabajo puedes irte a casa. |
| 이 컴퓨터를 사용해도 돼요? | ¿Puedo usar este ordenador? |
| 이거 제가 먹어도 돼요? — 네, 먹어도 돼요. | ¿Puedo comerme esto? — Sí, cómetelo. |
| 숙제를 오늘까지 내야 해요? — 아니요, 내일 내도 돼요. | ¿Hay que entregar la tarea hoy? — No, puedes entregarla mañana. |

**주의 — `되다` ↔ `괜찮다`:** en lugar de `되다` puedes usar `괜찮다` (estar bien, no importar) con el mismo significado: `-아/어도 괜찮아요` suena incluso un poco más suave.

| Coreano | Español |
|---|---|
| 이거 먹어도 괜찮아요? | ¿Está bien si me como esto? |
| 먼저 퇴근해도 돼요? — 네, 일을 다 했으면 가도 괜찮아요. | ¿Puedo salir del trabajo antes? — Sí, si terminaste todo, está bien que te vayas. |

## Comparación con lo que ya sabes
- **vs. `-(으)ㄹ 수 있다`** (1B): `할 수 있어요` = capacidad o posibilidad ("sé/puedo hacerlo"). `해도 돼요` = permiso ("me está permitido"). En español ambos son "poder", ahí está la trampa: *¿Puedes nadar?* (수영할 수 있어요?) vs. *¿Puedo nadar aquí?* (여기서 수영해도 돼요?).
- **vs. `-아/어야 되다`** (obligación): son gemelos opuestos con el mismo `되다`. `가야 돼요` = tienes que ir · `가도 돼요` = puedes ir (si quieres) · `안 가도 돼요` = no hace falta que vayas.
- **vs. `-(으)면 안 되다`** (la siguiente nota): es su negación de prohibición. Pregunta con `-아도 돼요?` → respuesta "no" con `-(으)면 안 돼요`.
- La partícula `도` ya la conoces ("también/incluso"): literalmente "incluso si haces V, está bien".

## Errores típicos de hispanohablantes
- Escribir `되요` en vez de `돼요`: la forma conjugada siempre es **돼** (되 + 어).
- Responder que no con `안 -아도 돼요`: `안 가도 돼요` significa "no hace falta ir", NO "no puedes ir". Para prohibir: `가면 안 돼요`.
- Usar `-(으)ㄹ 수 있다` para pedir permiso: `들어갈 수 있어요?` suena a "¿es posible entrar?"; para permiso educado di `들어가도 돼요?`.
- Con 쓰다: `쓰어도` ✕ → `써도` ○ (irregular 으).

## Machete para Pedro
**Pedir permiso = `V-아/어도 돼요?` · Darlo = `네, V-아/어도 돼요/괜찮아요` · Negarlo = `아니요, V-(으)면 안 돼요`**

가다 → 가도 돼요 · 먹다 → 먹어도 돼요 · 하다 → 해도 돼요
쓰다 → 써도 돼요 · 찍다 → 찍어도 돼요 · 앉다 → 앉아도 돼요

Y el combo obligación/permiso: `가야 돼요` (debes) / `가도 돼요` (puedes) / `안 가도 돼요` (no hace falta) / `가면 안 돼요` (prohibido).

## Mini-drill
1. `창문을 열어도 돼요?` — ¿Puedo abrir la ventana?
2. `이 옷을 입어 봐도 돼요?` — ¿Me puedo probar esta ropa? (combo con -아/어 보다)
3. `숙제는 내일까지 내도 돼요.` — La tarea se puede entregar hasta mañana.
4. `화장실 좀 써도 될까요?` — ¿Podría usar el baño? (más suave con -(으)ㄹ까요)
5. `사전을 안 가져와도 돼요.` — No hace falta que traigas el diccionario.

# Grammar 4 - V-(으)면 안 되다

Unidad: [[../Index - Unidad 14|Unidad 14: 여기서 사진을 찍어도 돼요?]]
Fuente: Student Book 2B (p. 123) + Appendix 문법 해설 (p. 247) + audios de la app koreano.

## La idea clave
`V-(으)면 안 되다` = **prohibición / negar permiso**: "no se puede...", "no se debe...", "está prohibido...".

Literalmente: `-(으)면` (si..., condicional que ya conoces) + `안 되다` (no está bien) → **"si haces V, no está bien"**. De ahí el sentido de regla o prohibición.

Es la pareja opuesta de [[V-아도 어도 되다|V-아도/어도 되다]]:
- `찍어도 돼요?` → `아니요, 찍으면 안 돼요.` (¿Puedo sacar fotos? — No, no se puede.)

Se usa para reglas de lugares (museo, biblioteca, residencia), normas de etiqueta (예절) y advertencias del médico, exactamente los temas de esta unidad.

## Forma
Se combina con **verbos** (con negaciones de estado se oye también con adjetivos en coloquial, pero el libro lo da solo con verbos):

| Raíz | Regla | Ejemplo |
|---|---|---|
| Sin 받침 | + 면 안 되다 | 보다 → **보면 안 되다** |
| Con 받침 | + 으면 안 되다 | 먹다 → **먹으면 안 되다** |
| Irregular ㄷ | ㄷ → ㄹ + 으면 | 듣다 → **들으면 안 되다** |
| Irregular ㅂ | ㅂ → 우 | 눕다 → **누우면 안 되다** |
| Terminada en ㄹ | + 면 (la ㄹ no cae) | 놀다 → **놀면 안 되다** |

Formalidad: `-(으)면 안 됩니다` (carteles, avisos formales), `-(으)면 안 돼요` (estándar), `-(으)면 안 돼` (반말).

## Appendix del libro (문법 해설)
**Significado:** expresa que **no se da permiso o aprobación** para una acción ("'-(으)면 안 되다' expresses denying permission or approval for an action").

**Combinación:** con raíces de verbo.

| | 받침 ✕ | 받침 ○ |
|---|---|---|
| 동사 | 보다 → **보면** 안 되다 | 먹다 → **먹으면** 안 되다 |

Ejemplos del appendix:

| Coreano | Español |
|---|---|
| 시험 시간에 사전을 보면 안 됩니다. | Durante el examen no se puede mirar el diccionario. |
| 여기에서 사진을 찍으면 안 돼요. | Aquí no se pueden sacar fotos. |
| 도서관에서 큰 소리로 떠들면 안 됩니다. | En la biblioteca no se puede armar bulla en voz alta. |

**주의 — doble negación `-지 않으면 안 되다`:** "si NO haces V, no está bien" = **obligación reforzada**. Significa lo mismo que `-아야/어야 되다` (tener que), pero enfatiza más el deber. Es como el español "no puedes no ir" / "tienes que ir sí o sí".

| Coreano | Español |
|---|---|
| 지금 가지 않으면 안 됩니다. → 지금 꼭 가야 합니다. | Si no vas ahora, mal. = Tienes que ir ahora sí o sí. |

## Comparación con lo que ya sabes
- **vs. `-아/어도 되다`**: permiso ↔ prohibición. Memorízalos como pareja pregunta-respuesta.
- **vs. `안 + V` / `-지 않다`**: `안 가요` solo niega ("no voy"); `가면 안 돼요` prohíbe ("no se puede ir").
- **vs. `-지 마세요`** (1B): `찍지 마세요` es una orden directa a alguien ("no saque fotos"); `찍으면 안 돼요` enuncia la regla general ("no está permitido sacar fotos"). Para explicar normas de un país o lugar, usa `-(으)면 안 되다`.
- **vs. `-아야/어야 되다`**: obligación. Fíjate en el sistema completo de 되다 que cierra esta unidad:

| Forma | Significado |
|---|---|
| V-아/어도 되다 | se puede (permiso) |
| V-(으)면 안 되다 | no se puede (prohibición) |
| V-아야/어야 되다 | hay que (obligación) |
| 안 V-아/어도 되다 | no hace falta (exención) |
| V-지 않으면 안 되다 | hay que sí o sí (obligación enfática) |

## Errores típicos de hispanohablantes
- Colocar mal el `안`: va pegado a `되다`, no al verbo. `안 먹으면 되다` significa otra cosa ("basta con no comer"). Prohibición = `먹으면 **안** 돼요`.
- Escribir `되요/안 되요`: siempre `돼요 / 안 돼요` cuando está conjugado.
- Confundir `안 가도 돼요` (no hace falta ir) con `가면 안 돼요` (prohibido ir). El primero exime, el segundo prohíbe.
- Traducir "no debes" siempre como `-(으)면 안 되다`: para un consejo suave existe `-지 않는 게 좋아요` (mejor no...); `-(으)면 안 되다` es regla dura.
- Con raíces en ㄹ (놀다, 떠들다) no se añade 으: `놀으면` ✕ → `놀면` ○.

## Machete para Pedro
**Regla/prohibición = `V-(으)면 안 돼요`. Piensa: "si V → mal".**

보다 → 보면 안 돼요 · 먹다 → 먹으면 안 돼요
피우다 → 피우면 안 돼요 · 찍다 → 찍으면 안 돼요
떠들다 → 떠들면 안 돼요 (ㄹ: sin 으)

Respuesta rápida: `-아도 돼요?` → sí: `네, -아도 돼요` / no: `아니요, -(으)면 안 돼요`.

## Mini-drill
1. `박물관에서 사진을 찍으면 안 됩니다.` — En el museo no se pueden sacar fotos.
2. `수업 시간에 휴대 전화를 사용하면 안 돼요.` — En clase no se puede usar el móvil.
3. `한국에서는 어른 이름을 부르면 안 돼요.` — En Corea no se debe llamar a los mayores por su nombre.
4. `약속에 늦으면 안 돼요.` — No puedes llegar tarde a la cita.
5. `내일까지 숙제를 내지 않으면 안 돼요.` — Tienes que entregar la tarea mañana sí o sí. (doble negación)

## Macheteado para repetir
- 한국어는 참 어려운 것 같아.
  - El coreano parece muy difícil.
  - Machete: 한국어는 / 참 / 어려운 / 것 / 같아.
- 너 한국어 잘하는데 왜?
  - Hablas bien el coreano, ¿por qué dices eso?
  - Machete: 너 / 한국어 / 잘하는데 / 왜?
- 아니야. 오늘 병원에 갔을 때 한국어를 잘못해서 창피한 일이 있었어.
  - No. Hoy cuando fui al hospital, hubo algo vergonzoso por no saber bien el coreano.
  - Machete: 아니야. / 오늘 / 병원에 / 갔을 / 때 / 한국어를 / 잘못해서 / 창피한 / 일이 / 있었어.
- 무슨 일이 있었는데?
  - ¿Qué pasó?
  - Machete: 무슨 / 일이 / 있었는데?
- 의사 선생님을 의사님이라고 불러서 사람들이 웃었어.
  - Llamé 의사님 al médico y la gente se rió.
  - Machete: 의사 / 선생님을 / 의사님이라고 / 불러서 / 사람들이 / 웃었어.

- 너도 한국어 때문에 실수한 적 있어?
  - ¿Tú también has cometido errores por el coreano?
  - Machete: 너도 / 한국어 / 때문에 / 실수한 / 적 / 있어?
- 물론이지. 그래도 실수하면서 배운 것은 안 잊어버려서 좋아.
  - Claro. Pero lo bueno es que lo que aprendes equivocándote no se olvida.
  - Machete: 물론이지. / 그래도 / 실수하면서 / 배운 / 것은 / 안 / 잊어버려서 / 좋아.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

## Macheteado para repetir
- 한국은 어른에 대한 예절이 좀 복잡한 것 같아요.
  - Parece que las normas de etiqueta para los mayores en Corea son un poco complicadas.
  - Machete: 한국은 / 어른에 / 대한 / 예절이 / 좀 / 복잡한 / 것 / 같아요.
- 맞아요. 어른 앞에서 하면 안 되는 것이 많지요?
  - Tienes razón. Hay muchas cosas que no se pueden hacer delante de los mayores.
  - Machete: 맞아요. / 어른 / 앞에서 / 하면 / 안 / 되는 / 것이 / 많지요?
- 네. 오늘 할아버지 앞에서 다리를 꼬고 앉아서 야단 맞았어요.
  - Sí. Hoy me regañaron por sentarme con las piernas cruzadas delante de mi abuelo.
  - Machete: 네. / 오늘 / 할아버지 / 앞에서 / 다리를 / 꼬고 / 앉아서 / 야단 / 맞았어요.
- 한국에서는 어른 앞에서 다리를 꼬고 앉으면 안 돼요.
  - En Corea no se puede sentar con las piernas cruzadas delante de los mayores.
  - Machete: 한국에서는 / 어른 / 앞에서 / 다리를 / 꼬고 / 앉으면 / 안 / 돼요.
- 저도 아는데 우리나라에서는 그렇게 해도 괜찮아서 자꾸 잊어버려요.
  - Lo sé, pero en mi país está bien hacerlo así y se me olvida.
  - Machete: 저도 / 아는데 / 우리나라에서는 / 그렇게 / 해도 / 괜찮아서 / 자꾸 / 잊어버려요.
- 일본 문화는 한국 문화와 많이 비슷하지요?
  - La cultura japonesa es muy parecida a la cultura coreana, ¿verdad?
  - Machete: 일본 / 문화는 / 한국 / 문화와 / 많이 / 비슷하지요?
- 비슷한 것도 있지만 다른 것도 많아요.
  - Hay cosas parecidas, pero también hay muchas cosas diferentes.
  - Machete: 비슷한 / 것도 / 있지만 / 다른 / 것도 / 많아요.
- 뭐가 다른데요?
  - ¿Qué es diferente?
  - Machete: 뭐가 / 다른데요?
- 한국에서는 어른 앞에서 술을 마실 때 고개를 돌리고 마셔야 하지만 일본에서는 그냥 마셔도 돼요.
  - En Corea, cuando se bebe alcohol delante de un mayor, hay que girar la cabeza, pero en Japón se puede beber directamente.
  - Machete: 한국에서는 / 어른 / 앞에서 / 술을 / 마실 / 때 / 고개를 / 돌리고 / 마셔야 / 하지만 / 일본에서는 / 그냥 / 마셔도 / 돼요.
- 그래요?
  - ¿Ah, sí?
  - Machete: 그래요?
- 가까운 나라지만 문화는 다르네요.
  - Son países cercanos, pero la cultura es diferente.
  - Machete: 가까운 / 나라지만 / 문화는 / 다르네요.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

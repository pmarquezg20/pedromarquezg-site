## Macheteado para repetir
- 잘 듣고 따라해 보세요.
  - Escuche bien y repita.
  - Machete: 잘 / 듣고 / 따라해 / 보세요.
- 1. 주말에는 극장에 사람이 많아요.
  - 1. Los fines de semana hay mucha gente en el cine.
  - Machete: 1. / 주말에는 / 극장에 / 사람이 / 많아요.
- 2. 기분이 좀 괜찮아졌어요?
  - 2. ¿Ya se siente un poco mejor?
  - Machete: 2. / 기분이 / 좀 / 괜찮아졌어요?
- 3. 언니가 아기를 낳았어요.
  - 3. Mi hermana mayor tuvo un bebé.
  - Machete: 3. / 언니가 / 아기를 / 낳았어요.
- 4. 언제 라면을 넣어요?
  - 4. ¿Cuándo pongo los fideos ramen?
  - Machete: 4. / 언제 / 라면을 / 넣어요?
- 물이 끓으면 넣으세요.
  - Póngalos cuando hierva el agua.
  - Machete: 물이 / 끓으면 / 넣으세요.

## Drill critico
- Escucha una vez sin leer y di en espanol que paso.
- Lee el coreano en voz alta marcando particulas: 은/는, 이/가, 을/를, 에/에서.
- Tapa la traduccion y reconstruye el dialogo usando solo el machete.

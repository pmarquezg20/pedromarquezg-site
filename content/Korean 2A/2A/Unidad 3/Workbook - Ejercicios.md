## Pág. 56 — Corregir la parte subrayada

**Instrucción:** 밑줄 친 부분을 맞게 고쳐 쓰세요 = "Corrige la parte subrayada"

**Regla clave — V-(으)러 vs V-(으)려고:**
- **-러** → solo con verbos de movimiento (가다, 오다, 다니다) — "voy *para* hacer algo"
- **-려고** → con cualquier verbo — "tengo intención de hacer algo"
- Truco: ¿el verbo principal es 가다/오다? → -러. Si no → -려고.

---

**1.** 저녁에 같이 밥 ~~먹으려고~~ 갑니다
→ **먹으러 갑니다**
→ 갑니다 = ir (movimiento) ✓ → necesita -러

**2.** 이번 주말에 ~~읽으러~~ 책을 샀어요
→ **읽으려고** 책을 샀어요
→ 샀어요 = comprar (no es movimiento) → necesita -려고

## Pág. 56 — Modificador nominal (pasado/presente)

**지연 씨, 이거 언제 ( ) 사진이에요?**
→ **2. 찍은** ✓ — la foto ya fue tomada → pasado → V-(으)ㄴ N

- 찍는 → presente: "la foto que estás tomando ahora" ✗
- 찍은 → pasado: "la foto que tomaste" ✓
- 찍는데 → conector de contexto, no modifica sustantivos ✗

Regla: ㄴ=pasado / 는=presente / ㄹ=futuro

## -러 vs -려고 con 다니다 (corrección)

Ambas son correctas y naturales:

> 한국어를 배우**려고** 언어교육원에 다녀요 ✓ (énfasis en la intención)
> 한국어를 배우**러** 언어교육원에 다녀요 ✓ (énfasis en la acción de ir)

**-(으)러** → solo con verbos de movimiento: 가다, 오다, **다니다**
**-(으)려고** → con cualquier verbo, expresa propósito o intención

La diferencia en esta frase es mínima. El libro usa -려고 porque es más general y reutilizable en más contextos. No es que -러 + 다니다 esté mal — está perfectamente bien.

> 💡 Regla práctica: si el verbo principal es 가다/오다/다니다 → puedes usar ambos. Si no es verbo de movimiento → solo -려고.

## Regla — orden fijo de V-러 가다/오다

**빵을 사러 가요** ✓
**빵을 가러 사요** ✗ — el orden es siempre: V(propósito)-러 + V(movimiento)

- 가러 no existe como conjugación de 가다
- el verbo de movimiento siempre va al final

> 사러 가요 = "voy a comprar" (lit. para-comprar voy)
> 그리러 나가요 = "salgo a dibujar"
> 먹으러 와요 = "viene a comer"

## Vocab — 랑

**랑** = "con" (casual) → igual que 하고 / 와/과 pero más informal entre amigos
→ 친구랑 = con un amigo

## Pág. 56 — -는데 en pasado

**어제 뭐 했어요? 친구랑 영화를 ( ) 재미있었어요.**
→ **봤는데** ✓ (opción correcta)
→ 봐는데 ✗ — esa forma no existe

Para -는데 en pasado: primero conjugar en pasado (-았/었-), luego añadir -는데
- 보다 → 봤어요 → 봤**는데** ✓

## Pág. 56 — Elegir el conector correcto

**쉬는 시간에 뭐 해요? 화장실에 ( ) 친구와 이야기해요.**
→ **3. 가거나** ✓

- 가거나 = "o" entre verbos → lista dos opciones ✓
- 가는데 = contexto suave → no sirve para listar opciones
- 가지만 = "pero" → contradicción
- 가려고 = "para" + movimiento → no tiene sentido aquí

## Pág. 56 — Dirección de 빌려주다

**사전을 안 가져왔는데 좀 (빌려줄까요)?** ✗
→ **빌려주세요** ✓ / **빌려줄 수 있어요?** ✓

- 빌려줄까요 = "¿te presto yo?" → el hablante ofrece prestar ✗
- 빌려주세요 = "préstame" → el hablante pide que le presten ✓
- La respuesta 네, 여기 있어요 confirma que el otro presta → el hablante debe pedir, no ofrecer

## Regla clave — -아/어서 NO con sugerencias ni mandatos

**날씨가 좋아서 산에 갈까요? ✗**
→ **날씨가 좋은데 산에 갈까요? ✓**

-아/어서 es causa directa. En coreano **no puede ir antes de:**
- -ㄹ까요? (¿vamos...?)
- -ㅂ시다 (¡vamos!)
- -(으)세요 (mandato)
- cualquier sugerencia

**Regla:** -아/어서 + ¿propuesta? → siempre mal. Usar **-는데/-ㄴ데** en su lugar.

> 날씨가 **좋은데** 산에 갈까요? ✓ *(Hace buen tiempo — ¿vamos al monte?)*

## Pizarra — 은데 vs 는데 (Junio 9)

| Tipo de palabra | Terminación          | Ejemplo          |
| --------------- | -------------------- | ---------------- |
| Verbo           | **-는데** (siempre)    | 배우다 → 배우**는데**   |
| Adjetivo        | **-(으)ㄴ데**           | 작다 → 작**은데**     |
| Pasado (V o A)  | **-았/었는데**           | 배웠다 → 배웠**는데**   |
| N이다             | **-인데**              | 짝이다 → 짝**인데**    |
| 있다 / 없다         | **-는데** (como verbo) | 재미있다 → 재미있**는데** |

### Ejemplos

- 배우다 (V) → 배우**는데**
- 배웠다 (pasado) → 배웠**는데**
- 재미있다 (A — pero 있다 ac-túa como verbo para -는데) → 재미있**는데**
- 작다 (A) → 작**은데**
- 짝이다 (N이다) → 짝**인데**

> 💡 Regla especial: 있다 / 없다 siempre usan **-는데** aunque sean adjetivos de estado.

## Pizarra — Los 3 modificadores nominales (Junio 9)

| Tiempo | Modificador | Ejemplo |
|---|---|---|
| Pasado (어제) | **V-(으)ㄴ** | 간 곳, 먹은 음식 |
| Presente (오늘) | **V-는** | 가는 곳, 먹는 음식 |
| Futuro (내일) | **V-(으)ㄹ** | 갈 곳, 먹을 음식 |

> La línea temporal: **ㄴ** (pasado) → **는** (presente) → **ㄹ** (futuro)

## 선생님 전화번호를 ___ 좀 가르쳐 주세요 (모르다)

→ **모르는데** ✓

- 모르다 = verbo → usa **-는데**
- Estructura: "No sé el número (contexto) — por favor dígame (petición)"
- -는데 aquí introduce contexto antes de una petición

---

## 싸고 예쁜 옷을 ___ 어디에 가면 좋아요? (사고 싶다)

→ **사고 싶은데** ✓

- 싶다 = "querer" — se comporta como **adjetivo**
- → usa **-(으)ㄴ데** (no -는데)
- 사고 싶은데 = "quiero comprar — (¿dónde me recomiendas?)"

---

## 싶다 = adjetivo, no acción

| Uso | Estructura | Ejemplo |
|---|---|---|
| Con -는데 (ERROR) | ~~사고 싶는데~~ | ✗ No existe |
| Con -(으)ㄴ데 (CORRECTO) | 사고 **싶은데** | ✓ |
| Modificador nominal | V-고 싶은 + N | 먹고 싶은 것, 가고 싶은 곳, 사고 싶은 옷 |

> 💡 싶다 siempre actúa como adjetivo: usa -(으)ㄴ데, -(으)ㄴ N, etc.
> - 먹고 싶은데 / 가고 싶은데 / 사고 싶은데 ✓
> - 먹고 싶은 것 / 가고 싶은 곳 / 사고 싶은 옷 ✓

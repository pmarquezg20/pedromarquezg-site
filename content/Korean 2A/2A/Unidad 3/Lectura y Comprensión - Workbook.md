## Unidad 3 — 콘서트에 가 봤어요?

### Texto — Workbook pág. 56 (preguntas 17–18)

> **A:** 지연 씨, 이거 언제 ( ㄱ ) 사진이에요?
> **B:** 지난 주말에 놀이공원에 가서 찍었어요.
> **A:** 재미있었어요?
> **B:** 네, 아주 재미있었어요. 그런데 마리코 씨는 주말에 ( ㄴ ) 뭐 해요?
> **A:** 저는 쇼핑을 자주 해요.

**Traducción:**
> **A:** Jiyeon, ¿cuándo fue tomada esta foto?
> **B:** Fui al parque de atracciones el fin de semana pasado y la tomé allí.
> **A:** ¿Fue divertido?
> **B:** Sí, fue muy divertido. Por cierto, Mariko, ¿qué haces normalmente los fines de semana?
> **A:** Yo voy de compras frecuentemente.

---

**17. ㄱ에 알맞은 것을 고르세요** *(Elige lo correcto para ㄱ)*

① 찍을 ② 찍는 **③ 찍은** ④ 찍는데

→ **③ 찍은** ✓
→ La foto ya fue tomada → pasado → V-(으)ㄴ N
→ 찍을 = futuro / 찍는 = presente / 찍는데 = conector, no modifica sustantivos

---

**18. ㄴ에 알맞은 것을 고르세요** *(Elige lo correcto para ㄴ)*

① 매일 **② 보통** ③ 별로 ④ 전혀

→ **② 보통** ✓ (normalmente)
→ A responde con 자주 (frecuentemente) → respuesta positiva → 보통 encaja
→ 별로/전혀 son frecuencias negativas → contradicen la respuesta
→ 매일 (cada día) sería redundante con 자주

### Texto — Workbook pág. 56 (preguntas 19–20)

> 저는 얼마 전에 그림 ( ㄱ )에 가입했습니다. 우리는 매주 토요일에 만나서 같이 그림을 그리러 나갑니다. 경치가 아름다운 곳에 가서 그림을 그리면 기분이 좋습니다. 혼자 ( ㄴ )도 좋지만 모임에 가면 취미가 같은 사람들을 만날 수 있어서 좋습니다.

**Traducción:**
> Hace poco me uní a un ( ㄱ ) de dibujo. Nos reunimos todos los sábados y salimos a dibujar juntos. Se siente bien ir a lugares con paisajes bonitos a dibujar. ( ㄴ ) solo también está bien, pero es bueno ir a las reuniones porque puedes conocer gente con el mismo hobby.

**Vocab nuevo:**
- **그리러 나갑니다** — 그리다 (dibujar) + -러 + 나가다 (salir) = "salimos a dibujar" *(V-러 con verbo de movimiento)*
- **경치** — paisaje / vista
- **아름답다** — ser bonito/hermoso → 아름다운 (modif.) *(ㅂ irregular)*
- **모임** — reunión / grupo
- **가입하다** — unirse / inscribirse

---

**19. ㄱ에 알맞은 것을 고르세요**

① 축제 ② 교회 **③ 동호회** ④ 전시회

→ **③ 동호회** ✓ (club de hobby/aficionados)
→ El texto habla de reunirse cada semana y salir a dibujar juntos → es un club
→ 축제=festival / 교회=iglesia / 전시회=exposición → no encajan

---

**20. ㄴ에 알맞은 것을 고르세요**

① 그릴 것 ② 그릴 줄 ③ 그린 것 **④ 그리는 것**

→ **④ 그리는 것** ✓ (el acto de dibujar — V-는 것, nominalización presente)
→ "혼자 그리는 것도 좋지만" = "dibujar solo también está bien, pero..."
→ 그릴 것=algo que dibujará (futuro) / 그릴 줄=saber dibujar / 그린 것=algo que dibujó (pasado)

### Texto del ejercicio (con blancos)

저는 다음 주에 약속이 많습니다. 월요일에 저녁 약속이 있습니다. 월요일에 [보기] (만날) 친구는 유진 씨입니다. 화요일에는 마리코 씨를 만나서 쇼핑을 1)( ) 계획입니다. 수요일에는 수영을 하러 가고 목요일에는 동호회 모임이 있습니다. 금요일에는 명동에 가고 토요일에는 친구들과 공연을 보러 갑니다. 친구들과 같이 2)( ) 공연은 사물놀이입니다. 일요일에는 여행을 가는데 이번에 3)( ) 곳은 경주입니다.

**Traducción:**
La semana que viene tengo muchos compromisos. El lunes tengo una cita para cenar. El amigo que (ejemplo: voy a encontrarme el lunes) es Yujin. El martes voy a encontrarme con Mariko, y es el plan 1)( ) de compras. El miércoles voy a nadar y el jueves hay una reunión del club. El viernes voy a Myeongdong y el sábado voy a ver una actuación con mis amigos. La actuación que voy a ver 2)( ) con mis amigos es el samulnori. El domingo voy de viaje, y el lugar al que voy a ir 3)( ) esta vez es Gyeongju.

### Respuestas

| # | Blanco | Respuesta | Estructura | Explicación |
|---|---|---|---|---|
| 보기 | 만날 | **만날** | V-(으)ㄹ N | futuro — "el amigo que voy a encontrar" |
| 1 | 쇼핑을 ___ 계획 | **하러** | V-(으)러 | propósito con verbo de movimiento implícito en 계획 |
| 2 | ___ 공연 | **볼** | V-(으)ㄹ N | futuro — "la actuación que voy a ver" |
| 3 | ___ 곳 | **갈** | V-(으)ㄹ N | futuro — "el lugar al que voy a ir" |

> 💡 Los tres blancos del ejercicio usan el modificador **V-(으)ㄹ N** porque todo es futuro — cosas que planeas hacer la semana que viene.

# 🎯 Drill #1 — Hablar: Unidad 5
*Producción oral — cronométrate. Meta: respuesta en menos de 5 segundos.*

---

## Bloque 1 — -(으)ㄹ까요? Suposición

Lee la situación → di la pregunta en coreano.

| Situación                                    | Tu frase             |
| -------------------------------------------- | -------------------- |
| ¿Lloverá mañana?                             | 내일 비가 올까요?           |
| ¿Le quedará grande esta ropa a Steven?       | 이 옷이 스티븐 씨에게 클까요?    |
| ¿Habrá paquetes para ir a Andong?            | 안동 가는 패키지여행이 있을까요?   |
| ¿Habrá llegado Nana a casa?                  | 나나 씨가 집에 도착했을까요?     |
| ¿Podrá el niño comer bien la comida picante? | 히엔 씨가 매운 음식을 잘 먹을까요? |
| ¿Qué será esto?                              | 이것은 무엇일까요?           |
| ¿Dónde sería bueno ir con mi amigo?          | 친구하고 어디에 가면 좋을까요?    |

## Bloque 2 — -(으)ㄹ 거예요 Predicción

Escucha la pregunta → responde con 아마 + -(으)ㄹ 거예요.

| Pregunta          | Tu respuesta |
| ----------------- | ------------ |
| 내일 비가 올까요?        | 아마 올 거예요     |
| 이 옷이 민수 씨에게 작을까요? | 아마 작을 거예요    |
| 친구가 안동을 좋아할까요?    | 아마 좋아할 거예요   |
| 수업이 끝났을까요?        | 네, 끝났을 거예요   |
| 오늘 나나 씨가 학교에 올까요? | 아마 못 올 거예요   |
| 패키지여행이 있을까요?      | 네, 있을 거예요    |

---

## Bloque 3 — -(으)니까 Causa → acción

Di la frase completa conectando la causa con lo que hay que hacer.

| Causa | Acción | Tu frase |
|---|---|---|
| 오늘 바쁘다 | 내일 만나다 (만납시다) | 오늘은 바쁘니까 내일 만납시다 |
| 기차가 버스보다 빠르다 | 기차를 타다 (타고 갑시다) | 기차가 버스보다 빠르니까 기차를 타고 갑시다 |
| 자세한 일정이 여기 있다 | 한번 보다 (보세요) | 자세한 일정이 여기 있으니까 한번 보세요 |
| 스티븐 씨는 책을 많이 읽다 | 책을 선물하다 (선물하세요) | 스티븐 씨는 책을 많이 읽으니까 책을 선물하세요 |
| 오늘 토요일이다 | 문을 일찍 닫다 (닫을 거예요) | 오늘은 토요일이니까 문을 일찍 닫을 거예요 |

---

## Bloque 4 — -고 나서 Secuencia

Conecta las dos acciones.

| Primero | Después | Tu frase |
|---|---|---|
| 이를 닦다 | 세수하다 | 이를 닦고 나서 세수해요 |
| 생각해 보다 | 연락드리다 | 생각해 보고 나서 연락드릴게요 |
| 한국어 공부를 마치다 | 대학원에 가다 | 한국어 공부를 마치고 나서 대학원에 가려고 해요 |
| 이 책을 다 읽다 | 친구에게 빌려주다 | 이 책을 다 읽고 나서 친구에게 빌려줄 거예요 |
| 밥을 먹다 | 물을 마시다 | 밥을 먹고 나서 물을 마셔요 |

## Bloque 5 — Role play rápido (agencia de viajes)

Practica en voz alta — tú eres 마리코.

**직원:** 어서 오세요. 어떻게 오셨어요?
**Tú:** *(Quiero ir a Jeonju 2 noches 3 días, ¿qué paquetes hay?)*
→ 전주에 2박 3일로 가려고 하는데 어떤 여행 상품이 있어요?

**직원:** 토요일에 출발하는 게 있어요. 요금은 20만 원이에요.
**Tú:** *(¿Qué incluye?)*
→ 뭐가 포함되어 있어요?

**직원:** 숙박비가 포함되어 있습니다.
**Tú:** *(Después de pensarlo, le llamo.)*
→ 생각해 보고 나서 연락드릴게요.

## Bloque 6 — Traducción rápida (español → coreano)

Sin mirar arriba. Cronométrate.

1. ¿Dónde sería bueno ir con mi novio?
2. Como el paisaje es bonito, vayamos a Andong.
3. Probablemente le gustará.
4. Después de pensarlo, le llamo.
5. ¿Habrá llegado ya?
6. Como mañana es festivo, llegará el lunes.
7. ¿Comerá bien el niño?
8. Creo que ya terminó la clase.

**Respuestas:**
1. 남자 친구하고 어디에 가면 좋을까요?
2. 경치도 아름다우니까 안동에 갑시다
3. 아마 좋아할 거예요
4. 생각해 보고 나서 연락드릴게요
5. 벌써 도착했을까요?
6. 내일이 휴일이니까 월요일에 도착할 거예요
7. 아이가 잘 먹을까요?
8. 수업이 끝났을 거예요

# Cheat Sheet — 문법 U1–U9
*Una línea por regla. Rápido y brutal.*

---

## U1 — 언어 교환

**N(이)라고 하다** — "se llama N"
→ 저는 페드로**라고** 해요. *(me llamo Pedro)*

**V-(으)려고** — "para / con intención de"
→ 공부**하려고** 왔어요. *(vine para estudiar)*

**V-거나** — "o" entre verbos
→ 읽**거나** 봐요. *(leo o veo)*

**N(이)나** — "o" entre sustantivos
→ 커피**나** 차 *(café o té)*

---

## U2 — 취미

**V-는 것** — nominalizar ("el hacer X")
→ 요리하**는 것**이 좋아요. *(me gusta cocinar)*

| Forma larga | Contracción | Función | Ejemplo                                       |
| ----------- | ----------- | ------- | --------------------------------------------- |
| -는 **것이**   | → **-는 게**  | sujeto  | 배우**는 게** 재미있어요 *(aprender es divertido)*     |
| -는 **것을**   | → **-는 걸**  | objeto  | 음악 듣**는 걸** 좋아해요 *(me gusta escuchar música)* |
| -는 **것은**   | → **-는 건**  | tema    | 말하**는 건** 어려워요 *(lo de hablar es difícil)*    |

**V-(으)ㄹ 줄 알다/모르다** — saber / no saber hacer
→ 수영**할 줄 몰라요**. *(no sé nadar)* 

**V-(으)ㄴ N** — modificador PASADO
→ 어제 **만든** 음식 *(la comida que hice ayer)*

**A/V-지 않다** — negación formal
→ 어렵**지 않아요**. *(no es difícil)*

---

## U3 — 경험

**V-아/어 보다** — experiencia / intentar
→ 먹어 **보세요**. *(pruébalo)* / 가 **봤어요**. *(fui/experiencia)*

**N 동안** — "durante N"
→ 일주일 **동안** *(durante una semana)*

**A-(으)ㄴ데 / V-는데** — dar contexto antes de decir algo
→ 배가 아**픈데** 약 있어요? *(me duele el estómago, ¿tienes medicina?)*

**N인데** — igual con sustantivos
→ 제 친구**인데** 같이 가요. *(es mi amigo, vamos juntos)*

**V-(으)ㄹ N** — modificador FUTURO
→ **줄** 선물 *(regalo que voy a dar)*

---

## U4 — 쇼핑

**A-(으)ㄴ/V-는/N인 것 같다** — "parece que..."
→ 좀 **큰 것 같아요**. *(parece un poco grande)*

**N보다 더** — "más que N"
→ 겨울**보다** 여름이 **더** 좋아요. *(me gusta más el verano que el invierno)*

**A/V-았/었으면 좋겠다** — "ojalá..."
→ 눈이 **왔으면 좋겠어요**. *(ojalá nieve)*

---

## U5 — 약속

**A/V-(으)ㄹ까요?** — "¿hacemos...? / ¿será...?"
→ 같이 **갈까요**? *(¿vamos juntos?)*

**A/V-(으)ㄹ 거예요** — va a... / creo que...
→ 내일 비가 **올 거예요**. *(creo que mañana lloverá)*

**A/V-(으)니까** — "porque" (razón que justifica)
→ 늦었**으니까** 빨리 가요. *(es tarde, así que vamos rápido)*

**V-고 나서** — "después de hacer"
→ 밥을 먹**고 나서** 공부해요. *(después de comer, estudio)*

---

## U6 — 우체국

**N(으)로** — hacia / por / con N (dirección, medio, herramienta)
→ 1번 출구**로** 나가세요. *(salga por la salida 1)*
→ 이메일**로** 보내요. *(lo mando por email)*

**N(이)라서** — "porque es N"
→ 학생**이라서** 할인이 돼요. *(como soy estudiante, hay descuento)*

**르 불규칙** — verbos irregulares en 르 (르→ㄹㄹ)
→ 모르다 → 몰라요 / 부르다 → 불러요

**V-(으)면 되다** — "basta con / es suficiente con"
→ 여기에 이름만 쓰**면 돼요**. *(basta con escribir el nombre aquí)*

**V-(으)ㄴ 것 같다** — "parece que + pasado"
→ 비가 **온 것 같아요**. *(parece que llovió)*

---

## U7 — 길 찾기

**A/V-(으)ㄹ 것 같다** — "parece que + futuro/suposición"
→ 비가 **올 것 같아요**. *(parece que va a llover)*

**V-는지 알다/모르다** — "saber/no saber si/cómo..."
→ 어디에 있**는지** 아세요? *(¿sabe dónde está?)*

**V-(으)려면** — "si quieres/para poder"
→ 한국어를 배우**려면** 매일 공부하세요. *(para aprender coreano, estudia cada día)*

**V-다가** — "estaba haciendo X y (cambio)"
→ 쭉 가**다가** 왼쪽으로 가세요. *(vaya recto y luego gire a la izquierda)*

---

## U8 — 날씨·계획

**A/V-겠-** — intención propia / suposición cortés
→ 제가 하**겠어요**. *(yo lo haré)* / 맛있**겠어요**. *(debe estar rico)*

**N 때문에** — "a causa de / por N"
→ 음식 **때문에** 힘들어요. *(es difícil por la comida)*

**V-아/어 버리다** — hacer del todo (con alivio o lamento)
→ 걱정을 잊어**버렸어요**. *(olvidé completamente las preocupaciones)*

**A/V-(으)ㄹ 때** — "cuando..."
→ 스트레스를 받**을 때** 뭐 해요? *(¿qué haces cuando te estresas?)*

---

## U9 — 상태·진행

**A-(으)ㄴ데요 / V-는데요** — dar info suavizando (tono de "¿te acuerdas?")
→ 지금 바쁜**데요**... *(es que ahora estoy ocupado...)*

**V-는 중이다 / N 중이다** — "estar en medio de"
→ 통화 **중이에요**. *(está en una llamada)* / 공부하**는 중이에요**. *(estoy estudiando)*

**A-(으)ㄴ가요 / V-나요** — pregunta suave/indirecta
→ 맛있**나요**? *(¿está rico?)*

**N밖에** — "solo N / nada más que N" (+ verbo negativo)
→ 천 원**밖에** 없어요. *(solo tengo mil wones)*

---

## ⚡ Los que más se confunden

| Regla      | Cuándo                          |
| ---------- | ------------------------------- |
| -(으)ㄴ N    | pasado → **만든** 음식              |
| -는 N       | presente → **만드는** 음식           |
| -(으)ㄹ N    | futuro → **만들** 음식              |
| -는 것 같다    | parece que ahora                |
| -(으)ㄴ 것 같다 | parece que pasó                 |
| -(으)ㄹ 것 같다 | parece que va a pasar           |
| -고 나서      | después de (secuencia)          |
| -고 나면      | después de (condición)          |
| -(으)니까     | porque (justifica lo que viene) |
| N 때문에      | a causa de N                    |

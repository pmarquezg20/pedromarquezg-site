# ⚡ Repaso Rápido — 문법 정리 (U1–U4)
*Todas las reglas en una página. Ejemplos del libro. Para repaso antes de clase y examen (중간시험 7/3).*

---

## Unidad 1 — 언어 교환

### 1. N(이)라고 하다 — "se llama / se dice"
**Para qué:** decir cómo se llama algo o presentarte.
- 저는 김민수**라고** 합니다. → *Me llamo Kim Minsu.* (vocal → 라고)
- 저는 히엔**이라고** 해요. → *Me llamo Hien.* (consonante → 이라고)

### 2. V-(으)려고 — "para / con la intención de"
**Para qué:** el propósito de una acción.
- 편지를 보내**려고** 우체국에 갔어요. → *Fui al correo para enviar una carta.*
- 코트 좀 보**려고** 왔어요. → *Vine a ver abrigos.* (¡aparece en Speaking U4!)

### 3. V-거나 — "o" (entre verbos)
- 일요일에는 책을 읽**거나** 영화를 봐요. → *Los domingos leo o veo películas.*

### 4. N(이)나 — "o" (entre sustantivos)
- 사과**나** 배를 좀 사 오세요. → *Compra una manzana o una pera.*
> Consonante → **이나** / vocal → **나**

---

## Unidad 2 — 취미가 뭐예요?

### 1. V-는 것 — nominalizar ("el hacer X")
**Para qué:** convertir un verbo en sustantivo (hobbies, gustos).
- 저는 사진 찍는 **것이** 취미예요. → *Mi hobby es tomar fotos.*
> Coloquial: 는 것이 → **는 게** / 는 것을 → **는 걸** / 는 것은 → **는 건**

### 2. V-(으)ㄹ 줄 알다/모르다 — "saber/no saber hacer"
- 저는 한글을 읽**을 줄 알아요**. → *Sé leer hangeul.*
- 민수 씨는 수영**할 줄 몰라요**. → *Minsu no sabe nadar.*
> Habilidad aprendida, no posibilidad (eso es -(으)ㄹ 수 있다).

### 3. V-(으)ㄴ N — modificador PASADO
- 어제 **만든** 음식은 김밥이에요. → *La comida que hice ayer es gimbap.*
- 그곳에서 **만난** 친구들 → *los amigos que conocí allí*
> Consonante → **은** / vocal → **ㄴ**

### 4. A/V-지 않다 — negación formal
- 별로 어렵**지 않았어요**. → *No fue muy difícil.*
- 오늘은 춥**지 않아요**. = 오늘은 **안** 추워요.
> -지 않다 = más formal/escrito; 안 = más hablado.

---

## Unidad 3 — 콘서트에 가 봤어요?

### 1. V-아/어 보다 — experiencia / intentar
- 작년 여름에 부산에 가 **봤어요**. → *El verano pasado fui a Busan (experiencia).*
- 김치찌개를 먹어 **보세요**. → *Prueba el kimchi jjigae (sugerencia).*
> **봤어요** = lo he hecho / **보세요** = inténtalo / **볼게요** = lo intentaré

### 2. N 동안 — "durante"
- 방학 **동안** 뭐 했어요? → *¿Qué hiciste durante las vacaciones?*
- 일주일 **동안** 영화제를 해요. → *Hacen el festival durante una semana.*
> Días nativos solo hasta 열흘 (10): 하루, 이틀, 사흘...

### 3. A-(으)ㄴ데 / V-는데 — dar contexto
**Para qué:** "X, y entonces / así que..." — prepara lo que viene.
- 지금 배가 좀 아**픈데** 약 있어요? → *Me duele el estómago, ¿tienes medicina?*
- 요즘 요리를 배우**는데** 재미있어요. → *Estoy aprendiendo a cocinar y es divertido.*
> ⚠️ Antes de sugerencias usa SIEMPRE -는데/-ㄴ데, NUNCA -아/어서.

### 4. N인데 — igual pero con sustantivos
- 제 친구**인데** 같은 학교에 다녀요. → *Es mi amigo — va a mi misma escuela.*
> Es 이다 + 는데 contraído.

### 5. V-(으)ㄹ N — modificador FUTURO
- 친구한테 **줄** 선물을 사러 가요. → *Voy a comprar un regalo que le daré a un amigo.*
- 구경**할 것**도 많습니다. → *También hay mucho que ver.* (lectura 부산)

## Unidad 4 — 옷이 좀 큰 것 같아요

### 1. 것 같다 — "parece que..."
**Para qué:** suposición + suavizar opiniones (cortesía).
- Adjetivo → **-(으)ㄴ 것 같다**: 기분이 좋**은 것 같아요** → *parece de buen humor*
- Verbo → **-는 것 같다**: 비가 오**는 것 같아요** → *parece que llueve*
- Sustantivo → **-인 것 같다**: 학생**인 것 같아요** → *parece estudiante*
- ⭐ 사이즈가 **좀 큰 것 같아요** → *la talla parece un poco grande* (frase de la unidad)
> 좀 + 것 같다 = el combo educado para opinar en tiendas.

### 2. N보다 — "más que N"
**Para qué:** comparar. Lo que pierde lleva 보다; lo que gana lleva 더.
- 구두**보다** 운동화가 **더** 편해요. → *Los tenis son más cómodos que los zapatos.*
- 저는 여름**보다** 겨울을 더 좋아해요. → *Me gusta más el invierno que el verano.*
> Intensificador: **훨씬** = muchísimo más (바다보다 산을 **훨씬** 더 좋아해요).

### 3. A/V-았/었으면 좋겠다 — "ojalá / me gustaría que"
**Para qué:** deseos y peticiones suaves.
- 사이즈가 좀 더 **컸으면 좋겠어요**. → *Ojalá la talla fuera más grande.*
- 크리스마스에 눈이 **왔으면 좋겠어요**. → *Ojalá nieve en Navidad.*
> Forma de pasado (-았/었) pero significado de deseo presente/futuro — como el subjuntivo "fuera".

### ⭐ Frase que une las 3 reglas de U4 (Speaking 2)
다른 거로 **바꿨으면 좋겠어요**... 색깔이 너무 **어두운 것 같아요**. 이것**보다** 좀 더 밝은 거 있어요?
→ *Me gustaría cambiarla... el color parece muy oscuro. ¿Hay una más clara que esta?*

---

## 🔑 Los 3 modificadores juntos (se confunden mucho)

- **PASADO** V-(으)ㄴ N → 어제 **본** 영화 *(la que vi)*
- **PRESENTE** V-는 N → 지금 **보는** 영화 *(la que veo)*
- **FUTURO** V-(으)ㄹ N → 내일 **볼** 영화 *(la que veré)*

## ⚠️ Errores que ya cometí (no repetir)

1. -아/어서 antes de sugerencia ✗ → usar **-는데/-ㄴ데**
2. 싶다 es ADJETIVO → 사고 **싶은데** ✓ (no 싶는데)
3. Días nativos solo hasta 열흘 — después sino-coreano

# Grammar 1-1 — A/V-(으)ㄹ 것 같다
📌 Suposición / predicción futura
🇰🇷 A/V-(으)ㄹ 것 같아요 → creo que (será/hará)... / parece que (va a)...
🇪🇸 Expresa suposición o predicción sobre algo futuro o no confirmado
🇬🇧 "I think it will..." / "It seems like it will..." — future supposition

---

## 형태 (Forma)

| Termina en | Forma |
|---|---|
| vocal o ㄹ | A/V + **ㄹ 것 같다** |
| consonante | A/V + **을 것 같다** |

**⚠️ Distinción clave con otras formas de 것 같다:**

| Tiempo | Forma | Ejemplo |
|---|---|---|
| Pasado (verbo) | V-(으)ㄴ 것 같다 | 먹은 것 같아요 |
| Presente (adjetivo) | A-(으)ㄴ 것 같다 | 바쁜 것 같아요 |
| Presente (verbo) | V-는 것 같다 | 먹는 것 같아요 |
| **Futuro/suposición** | **A/V-(으)ㄹ 것 같다** | **먹을 것 같아요** |

---

## 대화 (Track 63)

**아키라:** 나나 씨한테 무슨 선물이 좋**을까요**?
**유진:** 책이 좋**을 것 같아요**.

🔪 책(libro)이 좋을(será-bueno) 것(cosa) 같아요(parece)
*Creo que un libro sería bueno.*

---

## 책 예문 (Ejemplos del libro)

1. 이 옷은 좀 **클 것 같아요**.
   🔪 이(esta) 옷(ropa)은 좀(un-poco) 클(será-grande) 것 같아요(parece)
   *Creo que esta ropa está un poco grande.*

2. 내일 눈이 **올 것 같아요**.
   🔪 내일(mañana) 눈(nieve)이 올(vendrá) 것 같아요(parece)
   *Creo que mañana va a nevar.*

3. 이건 맵지 않아서 아이들도 잘 **먹을 것 같아요**.
   🔪 이건(esto) 맵지-않아서(como-no-es-picante) 아이들(niños)도 잘(bien) 먹을(comerán) 것 같아요(parece)
   *Como esto no es picante, creo que los niños también lo van a comer bien.*

---

## 문법 해설 (Notas oficiales del libro)

> A/V-(으)ㄹ 것 같다 expresa **predicción o suposición** sobre el futuro o sobre algo incierto.
> Se basa en la intuición o en indicios del hablante, no en certeza.

**Comparación con -(으)ㄹ 거예요:**
- -(으)ㄹ 거예요 → predicción/plan con más certeza
- -(으)ㄹ 것 같다 → suposición más tentativa, menos segura

**⚠️ -네요 combinado:** 을 것 같**네요** → descubrimiento espontáneo de la suposición.

---

## 연습1 — ¿Qué crees que pasará?

**보기:** 오후에 비가 올까요? → 네, 비가 **올 것 같아요**. 우산을 가져가세요.

1) 이 식당 음식이 맛있을까요? → 네, **맛있을 것 같아요**. 사람이 많아요.
2) 나나 씨 생일에 무슨 선물이 좋을까요? → 책이 **좋을 것 같아요**.
3) 내일 날씨가 어떨 것 같아요? → 오늘 구름이 많아서 비가 **올 것 같아요**.
4) 이 길이 복잡할까요? → 퇴근 시간이라서 좀 **막힐 것 같아요**.

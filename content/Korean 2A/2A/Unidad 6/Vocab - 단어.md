## 우체국 / Correo postal

| 한국어      | English               | Español                                                     |
| -------- | --------------------- | ----------------------------------------------------------- |
| 우표를 붙이다  | to put on a stamp     | pegar el sello — 우표(sello) + 붙이다(pegar)                     |
| 주소를 쓰다   | to write an address   | escribir la dirección — 주소(dirección postal) + 쓰다(escribir) |
| 봉투에 넣다   | to put in an envelope | meter en el sobre — 봉투(sobre) + 에(en) + 넣다(meter)           |
| 편지를 부치다  | to send a letter      | enviar una carta — 편지(carta) + 부치다(enviar correo)           |
| 엽서를 부치다  | to send a postcard    | enviar una postal — 엽서(postal) + 부치다                        |
| 소포를 포장하다 | to wrap a package     | empacar un paquete — 소포(paquete postal) + 포장하다(empacar)     |

> **부치다 vs 보내다:** 부치다 = enviar por correo (específico postal) / 보내다 = enviar en general
>
> **Orden lógico en el correo:** 봉투에 넣다 → 주소를 쓰다 → 우표를 붙이다 → 편지를 부치다

| 접수하다 | to accept; to receive | recibir / procesar |

# Grammar 2-3 — V-(으)ㄴ 것 같다
📌 Suposición sobre el pasado
🇰🇷 V-(으)ㄴ 것 같아요 → parece que V (ya ocurrió) / creo que V (pasado)
🇪🇸 Expresa suposición o deducción sobre algo que ya sucedió
🇬🇧 Expresses past supposition/inference — "it seems like (something) happened"

---

## 형태 (Forma)

| Verbo termina en | Forma |
|---|---|
| vocal | V + **ㄴ 것 같다** |
| consonante | V + **은 것 같다** |

> **Distinción con A-(으)ㄴ 것 같다** (U4): esa forma es para adjetivos en presente.
> Esta forma es para **verbos** → inferencia sobre acción ya completada.

---

## 대화 (Track 57)

**마리코:** 왜 이렇게 조용해요?
**지연:** 수업이 **끝난 것 같아요**. 학생들이 없네요.
**마리코:** 아키라 씨는 어디에 있어요?
**지연:** 벌써 점심을 **먹은 것 같아요**. 식당 쪽으로 갔거든요.

🔪 수업(clase)이 끝난(terminó) 것(cosa) 같아요(parece)
*Parece que la clase ya terminó.*

🔪 벌써(ya) 점심(almuerzo)을 먹은(comió) 것(cosa) 같아요(parece)
*Parece que ya almorzó.*

---

## 책 예문 (Ejemplos del libro)

1. 수업이 **끝난 것 같아요**.
   🔪 수업(clase)이 끝난(terminó) 것 같아요(parece)
   *Parece que la clase ya terminó.*

2. 아키라 씨는 벌써 점심을 **먹은 것 같아요**.
   🔪 아키라(Akira)씨는 벌써(ya) 점심(almuerzo)을 먹은(comió) 것 같아요(parece)
   *Parece que Akira ya almorzó.*

3. 지난주보다 환율이 **오른 것 같네요**.
   🔪 지난주(semana-pasada)보다 환율(tipo-de-cambio)이 오른(subió) 것 같네요(parece/ah-parece)
   *Parece que el tipo de cambio subió comparado con la semana pasada.*

---

## 문법 해설 (Notas oficiales del libro)

> V-(으)ㄴ 것 같다 = deducción/inferencia sobre acción **ya completada**.
> A diferencia de V-는 것 같다 (presente) o V-(으)ㄹ 것 같다 (futuro).

**Tres tiempos con 것 같다:**

| Tiempo | Forma | Ejemplo |
|---|---|---|
| **Pasado** (verbo) | V-(으)ㄴ 것 같다 | 먹은 것 같아요 |
| **Presente** (verbo) | V-는 것 같다 | 먹는 것 같아요 |
| **Presente** (adjetivo) | A-(으)ㄴ 것 같다 | 바쁜 것 같아요 |
| **Futuro** | V-(으)ㄹ 것 같다 | 먹을 것 같아요 |

**Contexto de uso:** Se basa en **evidencia indirecta** — lo que se ve, escucha u observa, pero no se sabe con certeza.

**-네요 con 것 같다:** 것 같네요 → suena más espontáneo/natural cuando descubres algo en el momento.

---

## 연습1 — ¿Qué pasó? Haz deducciones

*(Observa la situación y usa V-(으)ㄴ 것 같다)*

**보기:** 교실이 비어 있어요 → 수업이 끝난 것 같아요.

1) 친구가 자리에 없어요 → 밥을 **먹으러 간 것 같아요**
2) 가게 문이 닫혀 있어요 → 가게가 **문을 닫은 것 같아요**
3) 환율표가 바뀌었어요 → 환율이 **오른 것 같아요**
4) 이메일이 안 와요 → 아직 **못 본 것 같아요**

---

## Vocab del diálogo

**조용하다** — estar tranquilo/silencioso
**수업** — clase
**끝나다** — terminar → 끝난 (V-(으)ㄴ 형태)
**벌써** — ya (anteriormente a lo esperado)
**점심** — almuerzo
**식당** — restaurante, comedor
**환율** — tipo de cambio
**오르다** — subir → 오른 (르 불규칙)
**-네요** — partícula de descubrimiento/sorpresa espontánea

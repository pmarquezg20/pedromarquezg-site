# 문법 해설 — Grammar Reference
*Apéndice del libro: explicaciones detalladas por unidad*

---

## 1과 — 언어 교환

### 1. N(이)라고 하다
Se usa para citar o nombrar algo. "라고" se añade a citas o nombres.

**발침 X:** N + 라고 하다 → 스티븐**이라고** 합니다
**발침 O:** N + 이라고 하다 → 김민수**라고** 합니다

→ Ver [[Korean 2A/Unidad 1/Grammar 1-1 - N(이)라고 하다|Grammar 1-1 U1]]

---

### 2. V-(으)려고
Se añade al final de una cláusula para expresar intención o propósito.

**발침 X:** V + 려고 → 가**려고**
**발침 O:** V + 으려고 → 먹**으려고**

> 💡 **-(으)려고** vs **-(으)려고 하다**
> - **-(으)려고** sola: liga la intención con la acción siguiente en la misma frase
>   → 한국어를 배우**려고** 한국에 왔어요. *(Vine a Corea para aprender coreano.)*
> - **-(으)려고 하다** al final: la intención ES el predicado principal
>   → 한국에 가**려고 해요**. *(Tengo intención de ir a Corea.)*

→ Ver [[Korean 2A/Unidad 1/Grammar 1-2 - V-(으)려고|Grammar 1-2 U1]]

---

### 3. V-거나
Se añade a verbos para indicar opciones ("o... o..."). La raíz nunca cambia.

→ 주말에 영화를 보**거나** 음악을 들어요. *(Los fines de semana veo películas o escucho música.)*

→ Ver [[Korean 2A/Unidad 1/Grammar 2-1 - V-거나|Grammar 2-1 U1]]

---

### 4. N(이)나
Se añade a sustantivos para indicar opciones entre cosas ("o").

**발침 X:** N + 나 → 사과**나**
**발침 O:** N + 이나 → 책**이나**

→ Ver [[Korean 2A/Unidad 1/Grammar 2-2 - N(이)나|Grammar 2-2 U1]]

---

## 2과 — 취미가 뭐예요?

### 1. V-는 것
"-는 것" se añade a raíces verbales y convierte el verbo en frase nominal.

**발침 X:** 오다 → 오**는 것**
**발침 O:** 먹다 → 먹**는 것**

Puede funcionar como sujeto (con 이/가) o como objeto (con 을/를):
→ 운동하는 것**이** 건강에 좋아요. *(Hacer ejercicio es bueno para la salud.)*
→ 저는 운동하는 것**을** 좋아해요. *(Me gusta hacer ejercicio.)*

> 💡 En lengua hablada, "것" se abrevia con la partícula:
> **것이 → 게** / **것을 → 걸** / **것은 → 건**
> → 운동하는 **게** 좋아요 (más natural que 것이)

> 💡 **-기** reemplaza 것 en lengua hablada también:
> → 일찍 일어나는 것(→ **기**) 힘들어요.
> → 저는 영화 보는 것을(→ **걸**) 좋아해요.
> → 외국어를 배우는 것은(→ **건**) 재미있어요.

→ Ver [[Korean 2A/Unidad 2/Grammar 1-1 - V-는 것|Grammar 1-1 U2]]

---

### 2. V-(으)ㄹ 줄 알다/모르다
Se usa para expresar si sabes o no sabes hacer algo (habilidad).

**발침 X:** V + ㄹ 줄 알다 → 수영하**ㄹ** 줄 알아요
**발침 O:** V + 을 줄 알다 → 읽**을** 줄 알아요

> 💡 Diferencia con **V-(으)ㄹ 수 있다/없다** (posibilidad vs habilidad):
> - **줄 알다** = saber cómo (habilidad aprendida)
> - **수 있다** = poder (posibilidad según circunstancias)

→ Ver [[Korean 2A/Unidad 2/Grammar 1-2 - V-(으)ㄹ 줄 알다모르다|Grammar 1-2 U2]]

---

### 3. V-(으)ㄴ N
"(으)ㄴ" se añade a raíces verbales para modificar un sustantivo (pasado).

**발침 X:** 가다 → **간** N
**발침 O:** 먹다 → 먹**은** N
**ㄹ irregular:** 만들다 → 만**든** N

→ Ver [[Korean 2A/Unidad 2/Grammar 2-1 - V-(으)ㄴ N|Grammar 2-1 U2]]

---

### 4. A/V-지 않다
"-지 않다" se añade a raíces de adjetivos y verbos para negarlos.

**형용사:** 크다 → 크**지 않다** / 작다 → 작**지 않다**
**동사:** 가다 → 가**지 않다** / 먹다 → 먹**지 않다**

> 💡 **못 부정형** (otra forma de negación):
> "못 하다" = la forma de negación con 못 → 나는 한자를 읽지 못합니다
> = 어제는 바빠서 친구들을 만나지 못했어요

→ Ver [[Korean 2A/Unidad 2/Grammar 2-2 - A·V-지 않다|Grammar 2-2 U2]]

---

## 3과 — 콘서트에 가 봤어요?

### 1. V-아/어 보다
"-아/어 보다" se añade a raíces verbales para expresar experiencia pasada o intento.

**아/오 → 아 보다:** 가다 → 가 보다
**otras → 어 보다:** 먹다 → 먹어 보다
**하다 → 해 보다:** 요리하다 → 요리해 보다

→ 제주도에 가**봤어요**. *(He ido a Jeju.)*
→ 번지 점프를 해**보세요**. *(Prueba el bungee jumping.)*

→ Ver [[Korean 2A/Unidad 3/Grammar 1-1 - V-아어 보다|Grammar 1-1 U3]]

---

### 2. N 동안
"동안" se añade a sustantivos de tiempo para expresar duración.

**발침 X/O:** N + 동안 (la forma no cambia nunca)
→ 일주일 **동안** / 한 달 **동안** / 3년 **동안**

→ 얼마 동안 한국어를 공부했어요? *(¿Cuánto tiempo has estudiado coreano?)*
→ 방학 동안 제주도에 갔다 왔어요. *(Durante las vacaciones fui a Jeju.)*

> 💡 **Días nativos 1–10** (solo existen hasta 10, del libro p.239):
> 하루(1) 이틀(2) 사흘(3) 나흘(4) 닷새(5) 엿새(6) 이레(7) 여드레(8) 아흐레(9) 열흘(10)

→ Ver [[Korean 2A/Unidad 3/Grammar 1-2 - N 동안|Grammar 1-2 U3]]

---

### 3. A-(으)ㄴ데, V-는데 / N인데
Se usa para dar información de fondo/contexto antes de decir algo — contraste, trasfondo, o preámbulo a una pregunta.

**형용사:** A + (으)ㄴ데 → 바쁜데, 좋은데
**동사:** V + 는데 → 가는데, 먹는데
**명사:** N + 인데 → 학생인데

→ 저 학생**인데요**, 질문이 있어요. *(Soy estudiante, tengo una pregunta.)*
→ 내일 바**쁜데**, 같이 갈 수 있어요? *(Mañana estoy ocupado, ¿puedes ir conmigo?)*
→ 이 음식을 먹어 봤**는데**, 정말 맛있어요. *(He comido esta comida y está muy rica.)*
→ 이제 집에 가**는데**, 오늘 재미있었어요. *(Me voy a casa ya, hoy fue divertido.)*

> 💡 Muy versátil: puede ser contraste ("pero..."), contexto ("resulta que..."), o trasfondo para una pregunta. El tono lo decide el contexto.

→ Ver [[Korean 2A/Unidad 3/Grammar 2-1 - A-(으)ㄴ데 V-는데|Grammar 2-1 U3]] · [[Korean 2A/Unidad 3/Grammar 2-2 - N인데|Grammar 2-2 U3]]

---

### 4. V-(으)ㄹ N *(modificador nominal futuro)*
"(으)ㄹ" se añade a raíces verbales para modificar un sustantivo con sentido futuro o prospectivo.

**발침 X:** V + ㄹ + N → 갈 곳 *(el lugar al que iré)*
**발침 O:** V + 을 + N → 먹을 음식 *(la comida que voy a comer)*

→ 내일 **갈** 곳이 어디예요? *(¿Adónde vas a ir mañana?)*
→ 오늘 **먹을** 음식을 만들었어요. *(Preparé la comida que vamos a comer hoy.)*
→ 친구한테 **줄** 선물을 샀어요. *(Compré un regalo que le daré a un amigo.)*

> 💡 Resumen de los 3 modificadores nominales:
> **V-(으)ㄴ N** → pasado: 어제 먹**은** 음식 *(la comida que comí)*
> **V-는 N** → presente: 매일 먹**는** 음식 *(la comida que como)*
> **V-(으)ㄹ N** → futuro: 내일 먹**을** 음식 *(la comida que comeré)*

→ Ver [[Korean 2A/Unidad 3/Grammar 2-3 - V-(으)ㄹ N|Grammar 2-3 U3]]

---

### 5. V-(으)ㄴ 것 같다 *(pasado especulativo)*
Se usa para especular sobre eventos pasados basándose en percepciones.

**받침 X:** V + ㄴ 것 같다 → 간 것 같다
**받침 O:** V + 은 것 같다 → 먹은 것 같다

→ 스티븐 씨가 집에 **간 것 같아요**. 자리에 가방이 없어요.
→ 유진 씨가 그 옷을 선물로 **받은 것 같아요**.

---

## 4과 — 옷이 좀 큰 것 같아요

### 1. A-(으)ㄴ 것 같다, V-는 것 같다, N인 것 같다
여러 상황으로 미루어 현재 그런 일이 일어나거나 그러한 상태에 있다고 추측함을 나타낸다.
*(Expresa conjetura o suposición basada en el propio sentido, observación o juicio.)*

**형용사:** A + (으)ㄴ 것 같다 → 크다 → **큰 것 같다** / 작다 → **작은 것 같다**
**동사:** V + 는 것 같다 → 가다 → **가는 것 같다** / 먹다 → **먹는 것 같다**
**명사:** N + 인 것 같다 → 가수 → **가수인 것 같다** / 학생 → **학생인 것 같다**

예) 수미는 요즘 바쁜 **것 같아요**.
→ 피곤한 것 같은데 좀 쉬세요.
→ 지금 밖에 비가 오는 **것 같아요**.
→ 열집에서 불고기를 먹는 **것 같아요**. 불고기 냄새가 나요.
→ 저 사람은 지연 씨 동생**인 것 같아요**.

> 💡 '있다, 없다'로 끝나는 형용사는 '-는 것 같다'와 결합한다:
> → 저 식당이 싸**고** 맛있는 **것 같아요**.
> → 집에 전화를 했는데 안 받아요. 집에 아무도 없는 **것 같아요**.

> 💡 **Negación:** A/V-지 않은 것 같다 / A/V-지 않는 것 같다 / N이 아닌 것 같다

→ Ver [[Korean 2A/Unidad 4/Grammar 1-1 - 것 같다|Grammar 1-1 U4]]

---

### 2. N보다
앞말이 비교의 기준이 되는 대상임을 나타낸다.
*(When comparing two things, '보다' marks the basis of comparison — the counterpart of English "than".)*

**명사와 결합한다:** N + 보다 (sin cambio de forma)
→ 어제 → 어제**보다** / 월요일 → 월요일**보다**

예) 오늘이 어제**보다** 더워요.
→ 월요일**보다** 화요일이 더 바빠요.
→ 이 책이 그 책**보다** 어려워요.

> 💡 '보다'는 비교의 기준이 되는 '보다' 앞이나 뒤에 모두 올 수 있다:
> → 오늘이 어제**보다** 더 더워요. = 어제**보다** 오늘이 더 더워요.

→ Ver [[Korean 2A/Unidad 4/Grammar 2-1 - N보다|Grammar 2-1 U4]]

---

### 3. A/V-았/었으면 좋겠다
어떤 일이 이루어졌으면 하는 희망이나 바람을 나타낸다.
*(Expresses the speaker's hope or desire.)*

| | ㅏ, ㅗ | 하다 | ㅣ, ㅡ, ㅓ… |
|---|---|---|---|
| 형용사 | 작다 → 작았으면 좋겠다 | 편하다 → 편했으면 좋겠다 | 길다 → 길었으면 좋겠다 |
| 동사 | 가다 → 갔으면 좋겠다 | 좋아하다 → 좋아했으면 좋겠다 | 먹다 → 먹었으면 좋겠다 |

예) 이번 휴가 때는 여행을 갔으면 좋겠어요.
→ 그 친구가 나를 오래 기억**했으면** 좋겠어요.
→ 돈을 많이 벌었으면 좋겠어요.
→ 치마 색깔이 좀 더 밝았으면 좋겠어요.

> 💡 '-았/었으면 좋겠다'는 '-(으)면 좋겠다'와 바꿔 쓸 수 있다. 바라는 일이 이루어지기 힘들거나 실현 가능성이 적은 일일 경우 '-았/었으면 좋겠다'를 써서 바라는 일을 강조하는 경향이 있다:
> → 세계 여행을 하**면** 좋겠어요. (posible)
> → 세계 여행을 **했으면** 좋겠어요. (deseo más intenso, menos probable)

→ Ver [[Korean 2A/Unidad 4/Grammar 2-2 - 았었으면 좋겠다|Grammar 2-2 U4]]

## 5과 — 어디에 가면 좋을까요?

### 1. A/V-(으)ㄹ까요?
어떤 상황에 대한 추측이나 의심에 대해 상대의 생각을 물을 때 사용한다.
*(Used to ask for the listener's opinion about something unknown to or wondered about by the speaker.)*

**형용사:** A + (으)ㄹ까요? → 크다 → **클까요?** / 작다 → **작을까요?**
**동사:** V + (으)ㄹ까요? → 오다 → **올까요?** / 먹다 → **먹을까요?**

예) 지금 길이 복잡**할까요**?
→ 이 옷이 스티븐 씨에게 작**을까요**?
→ 내일 비가 **올까요**?
→ 이 음식을 아이가 잘 먹**을까요**?
→ 주말인데 식당을 예약할 수 있**을까요**?

→ Ver [[Korean 2A/Unidad 5/Grammar 1-1 - A-V-(으)ㄹ까요|Grammar 1-1 U5]]

---

### 2. A/V-(으)ㄹ 거예요
어떤 상황에 대한 추측이나 자신의 생각을 완곡하게 말할 때 사용한다.
*(Used to express one's assumption or opinion about a certain situation in a less direct way.)*

**형용사:** A + (으)ㄹ 거예요 → 크다 → **클 거예요** / 작다 → **작을 거예요**
**동사:** V + (으)ㄹ 거예요 → 오다 → **올 거예요** / 먹다 → **먹을 거예요**

예) 일찍 출발하세요. 길이 많이 막**힐 거예요**.
→ 이 옷이 스티븐 씨에게 맞**을까요**? — 네, 아마 맞**을 거예요**.
→ 아이가 이 음식을 잘 먹**을까요**? — 네, 아마 잘 먹**을 거예요**.

> 💡 '아마'를 자주 같이 써서 불확실성을 강조한다:
> → 내일 아마 비가 **올 거예요**. / 민수 씨가 아마 이 선물을 좋아**할 거예요**.

> 💡 추측 의미의 '-(으)ㄹ까요?'의 대답으로는 '-ㄹ 거예요'를 쓸 수 있지만, 제안을 나타내는 '-ㄹ까요?'의 대답으로는 쓸 수 없다:
> → 내일 비가 올까요? — 아마 **올 거예요**. ✓
> → 같이 영화 볼까요? — 네, 볼 거예요. ✗ / 네, 좋아요. 같이 봐요. ✓

→ Ver [[Korean 2A/Unidad 5/Grammar 1-2 - A-V-(으)ㄹ 거예요|Grammar 1-2 U5]]

---

### 3. A/V-(으)니까, N(이)니까
앞의 내용이 뒤에 오는 내용의 이유나 근거임을 나타낸다.
*(These forms indicate the preceding clause is the reason for or basis for the following clause.)*

**형용사:** 바쁘다 → 바쁘**니까** / 작다 → 작**으니까**
**동사:** 가다 → 가**니까** / 먹다 → 먹**으니까**
**명사:** 친구 → 친구**니까** / 일요일 → 일요일**이니까**

예) 불고기가 맛있**으니까** 불고기를 드세요.
→ 날씨가 너무 추**우니까** 밖에 나가지 마세요.
→ 집이 여기에서 머**니까** 빨리 가야 돼요.
→ 구두를 신고 걸**으니까** 발이 아파요.
→ 월요일**이니까** 길이 많이 막힐 거예요.

> 💡 **-(으)니까 vs -아/어서:**
> | -(으)니까 | -아/어서 |
> |---|---|
> | 후행절 제약 없음 (명령, 제안, 청유 가능) | 명령/제안 불가 |
> | 과거 '-았/었-' 결합 가능 | 과거 결합 불가 |
> | 날씨가 좋**으니까** 산에 갈까요? ✓ | 날씨가 좋**아서** 산에 갈까요? ✗ |

→ Ver [[Korean 2A/Unidad 5/Grammar 2-1 - A-V-(으)니까|Grammar 2-1 U5]]

---

### 4. V-고 나서
어떤 행위를 끝낸 다음에 다른 행위를 하거나 어떤 상황이 일어나게 되었음을 나타낸다.
*(Indicates that an action or event has been completed and another action or event follows.)*

**동사와 결합:** V + 고 나서 (forma invariable)
→ 가다 → 가**고 나서** / 먹다 → 먹**고 나서**

예) 부모님과 의논하**고 나서** 결정했어요.
→ 저녁 먹**고 나서** 같이 커피 마셔요.
→ 마리코 씨가 가**고 나서** 스티븐 씨가 왔어요.
→ 영화가 끝나**고 나서** 사람들이 밖으로 나갔어요.

→ Ver [[Korean 2A/Unidad 5/Grammar 2-2 - V-고 나서|Grammar 2-2 U5]]

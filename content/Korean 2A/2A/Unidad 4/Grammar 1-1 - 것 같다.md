# Grammar 1-1 — A-(으)ㄴ 것 같다, V-는 것 같다, N인 것 같다
*(문법과 표현 1 — pág. 94, Track 33)*

**📌 Lo que hace:** Dice que algo "parece" ser así — una suposición basada en lo que ves o sientes
**🇰🇷 Estructura:** adjetivo + (으)ㄴ 것 같다 / verbo + 는 것 같다 / sustantivo + 인 것 같다
**🇪🇸 Español:** "Parece que está estudiando" → 공부하**는 것 같아요**
**🇬🇧 English:** "It seems like he's studying" → 공부하**는 것 같아요**

---

## Forma

**Adjetivo** → **-(으)ㄴ 것 같다** — 좋다 → 좋**은 것 같아요** / 크다 → 큰 **것 같아요**
**Verbo (presente)** → **-는 것 같다** — 공부하다 → 공부하**는 것 같아요**
**Sustantivo** → **-인 것 같다** — 학생 → 학생**인 것 같아요**

💡 Es la forma educada de suavizar opiniones: en vez de "está grande" dices "parece un poco grande". Los coreanos lo usan muchísimo en tiendas y al opinar.

---

## Diálogo del libro

> A: 준호는 지금 뭐 해요?
> *(¿Qué hace Junho ahora?)*
>
> B: 공부하**는 것 같아요**.
> *(Parece que está estudiando.)*
> 🔪 *estudiar-cosa / igual-es* → "es igual a la cosa de estudiar" = parece que estudia
> *공부해요* (la acción) → *공부하는 것 같아요* (suposición — lo veo desde la puerta)

---

## Ejemplos del libro

> 유진 씨가 오늘 기분이 좋**은 것 같아요**.
> - 기분이 좋다 = estar de buen humor (adjetivo → -은 것 같다)
> - *"Parece que Yujin está de buen humor hoy."*
> 🔪 *Yujin / hoy / ánimo / buena-cosa / igual-es*

> 지금 밖에 비가 오**는 것 같아요**.
> - 비가 오다 = llover (verbo → -는 것 같다)
> - *"Parece que está lloviendo afuera ahora."*
> 🔪 *ahora / afuera-en / lluvia / venir-cosa / igual-es*

> 저 사람은 우리 학교 학생**인 것 같아요**.
> - 학생 = estudiante (sustantivo → -인 것 같다)
> - *"Esa persona parece ser estudiante de nuestra escuela."*
> 🔪 *esa / persona / nuestra / escuela / estudiante-ser-cosa / igual-es*

> 보기: 기분이 좋**은 것 같아요**.
> *(Parece que está de buen humor.)* — viendo a alguien silbar contento

---

## Las 3 formas en una frase cada una

**Adjetivo:** 사이즈가 좀 **큰 것 같아요** — *la talla parece un poco grande* (¡frase clave de la unidad!)
**Verbo:** 비가 **오는 것 같아요** — *parece que llueve*
**Sustantivo:** 학생**인 것 같아요** — *parece ser estudiante*

---

## Vocab útil

**기분** — estado de ánimo (기분이 좋다 = estar de buen humor)
**밖에** — afuera
**비가 오다** — llover
**저 사람** — esa persona

## Ejercicios

**1. Transforma en suposición (것 같다):**

1. 사이즈가 크다 → *parece que la talla es grande*
2. 비가 오다 → *parece que llueve*
3. 선생님이다 → *parece ser profesor/a*
4. 색깔이 어둡다 → *parece que el color es oscuro*

**2. Responde con 것 같다:**

저 사람은 지금 뭐 해요? *(está leyendo un libro)*

# Speaking 2 — 말하기 2 (Track 37)
*Unidad 4 — 옷이 좀 큰 것 같아요 · pág. 100*

*Reglas en uso: [[Korean 2A/Unidad 4/Grammar 1-1 - 것 같다|G1-1 것 같다]] · [[Korean 2A/Unidad 4/Grammar 2-1 - N보다|G2-1 N보다]] · [[Korean 2A/Unidad 4/Grammar 2-2 - 았었으면 좋겠다|G2-2 -았/었으면 좋겠다]] · [[Korean 2A/Unidad 3/Grammar 1-1 - V-아어 보다|V-아/어 보다 (U3)]]*

## Conversación

**점원**: 어서 오세요. 찾으시는 거 있으세요?
**아키라**: 어제 이 넥타이를 샀는데 다른 거로 **바꿨으면 좋겠어요**.
**점원**: 왜요? 마음에 안 드세요?
**아키라**: 네, 색깔이 너무 **어두운 것 같아요**. 이것**보다** 좀 더 밝은 거 있어요?
**점원**: 그럼 이건 어떠세요? 한번 **해 보세요**.

· · ·

**점원**: 잘 어울리시네요. 이거로 드릴까요?
**아키라**: 네, 이거로 주세요.

---

### Traducción

**점원**: Bienvenido. ¿Hay algo que esté buscando?
**아키라**: Ayer compré esta corbata y me gustaría cambiarla por otra.
**점원**: ¿Por qué? ¿No le gusta?
**아키라**: Sí (no me gusta), el color parece demasiado oscuro. ¿Hay alguna más clara que esta?
**점원**: Entonces ¿qué le parece esta? Pruébesela.

· · ·

**점원**: Le queda muy bien. ¿Le doy esta?
**아키라**: Sí, deme esta.

---

## Vocab

**어서 오세요** — bienvenido (saludo de tienda)
**샀는데** — compré, y... *(사다 + -았는데: contexto antes de pedir el cambio)*
**다른 거로 바꾸다** — cambiar por otro
**바꿨으면 좋겠어요** — me gustaría cambiarla *(G2-2 ✓ — petición suave)*
**마음에 안 드세요?** — ¿no le gusta?
**너무 어두운 것 같아요** — parece demasiado oscuro *(G1-1 ✓)*
**이것보다** — que este / comparado con este *(G2-1 ✓)*
**한번 해 보세요** — pruébesela *(corbata → 하다 + -아/어 보다)*
**잘 어울리시네요** — le queda muy bien *(어울리다 + -시 honorífico + -네요 admiración)*
**이거로 드릴까요?** — ¿le doy esta? *(드리다 = dar honorífico)*
**이거로 주세요** — deme esta

## 연습1 — Practica sustituyendo

Patrón: [prenda] 샀는데 바꿨으면 좋겠어요 → [problema] 것 같아요 → 이것보다 더 [deseo] 거 있어요? → 한번 [verbo de ponerse] 보세요

1. **넥타이** — 색깔이 너무 어둡다 → 밝다 → **하다**
2. **바지** — 길이가 좀 짧다 → 길다 → **입다**
3. **장갑** — 좀 작다 → 크다 → **끼다**
4. **구두** — 굽이 좀 높다 → 낮다 → **신다**

*Ejemplo con 3): 장갑을 샀는데 좀 작은 것 같아요. 이것보다 좀 더 큰 거 있어요? → 한번 껴 보세요.*

> ⚠️ Ojo: cada prenda usa SU verbo de ponerse (página de vocab 4과): 넥타이 → 하다, 바지 → 입다, 장갑 → 끼다, 구두 → 신다. Por eso la última línea cambia: 해 보세요 / 입어 보세요 / 껴 보세요 / 신어 보세요.

**굽이 높다/낮다** — el tacón es alto/bajo

> 💡 **바꿨으면 좋겠어요** en vez de 바꿔 주세요 — el deseo (G2-2) suena menos exigente que el imperativo. Igual que 것 같다, es cortesía.
> 💡 Las TRES gramáticas de la unidad aparecen en UNA frase de Akira: 색깔이 너무 어두운 **것 같아요** (G1-1). 이것**보다** 좀 더 밝은 거 있어요? (G2-1) — y antes 바꿨**으면 좋겠어요** (G2-2). Memoriza este diálogo y tienes la unidad entera.

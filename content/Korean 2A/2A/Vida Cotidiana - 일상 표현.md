# Vida Cotidiana — 일상 표현
*Expresiones reales para situaciones del día a día en Corea*

---

## En el subway/restaurante de sándwiches (como Subway)

| Situación | Coreano | Español |
|---|---|---|
| ¿Para comer aquí o para llevar? | 여기서 드실 건가요, 가져가실 건가요? | Expresión formal del empleado |
| Para comer aquí | 여기서 먹을게요 | |
| Para llevar | 포장해 주세요 | 포장 = empacar/envolver |
| Para llevar | 가져갈게요 | lit. "me lo llevaré" |
| Take out, por favor | 테이크아웃으로 주세요 | más informal |

> 💡 **포장** (empacar) es la palabra más común en tiendas y restaurantes coreanos para pedir para llevar.

---

## En tiendas y restaurantes

| Situación | Coreano | Nota |
|---|---|---|
| ¿Cuánto cuesta? | 얼마예요? | lo más básico |
| Esto, por favor | 이거 주세요 | señalando el artículo |
| Una más | 하나 더 주세요 | 더 = más |
| La cuenta, por favor | 계산해 주세요 | 계산 = cálculo/cuenta |
| ¿Tiene tarjeta? (¿acepta tarjeta?) | 카드 돼요? | muy casual y común |
| En efectivo | 현금으로 할게요 | 현금 = efectivo |
| Bolsa, por favor | 봉투 주세요 | 봉투 = bolsa/sobre |
| Recibo, por favor | 영수증 주세요 | |

## Frases útiles en conversación

| Situación | Coreano | Español |
|---|---|---|
| No entendí, repita por favor | 잘 못 들었는데 다시 말씀해 주세요 | lit. "no escuché bien — diga de nuevo por favor" |
| Más despacio, por favor | 천천히 말씀해 주세요 | 천천히 = despacio |
| ¿Cómo se dice esto en coreano? | 이게 한국어로 뭐예요? | para señalar algo |
| No entiendo | 이해가 안 돼요 | |
| ¿Puede escribirlo? | 써 주실 수 있어요? | para leer en vez de escuchar |
| Un momento, por favor | 잠깐만요 | muy común |
| Perdón / disculpe | 죄송합니다 / 실례합니다 | 죄송 = más formal, 실례 = al pasar |

## En el transporte público

| Situación | Coreano | Nota |
|---|---|---|
| ¿Para ir a ___? | ___에 가려면 어떻게 해요? | ¿cómo voy a...? |
| ¿Cuánto tarda? | 얼마나 걸려요? | |
| ¿Qué línea tomo? | 몇 호선을 타야 해요? | 호선 = línea de metro |
| Bajar en esta estación | 여기서 내려요 | |
| Transferencia | 환승 | señales en el metro |
